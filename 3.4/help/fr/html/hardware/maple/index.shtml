JMRI: Systèmes Maple


Maple Systems a fait plusieurs séries de panneaux écran tactile qui peut être très utile sur les chemins de fer modèle.

Chaque panneau est vraiment un ordinateur autonome. C'est configuré et chargé en utilisant un programme de configuration fournis par les systèmes Maple. Cette configuration définit les affichages, les étiquettes de bouton et de lieux, etc.

JMRI interagit avec le panneau d'entrée et par les bits de sortie (appelés «bobines», comme dans les bobines de relais) dans le panneau. Le panneau de configuration peut définir ces réponse à divers événements. JMRI fournit des objets Capteur qui les rendent disponibles pour déclencher des actions sans JMRI. JMRI peut définir les objets Aiguillages qui peuvent les régler, auquel cas le panneau de configuration peut changer ce qui apparaît.J

Ceci est encore au stade très précoce, donc les détails changent rapidement. Si vous souhaitez utiliser ces dispositifs, s'il vous plaît contacter les développeurs JMRI.

La lettre du système pour les connexions Maple est «K».

Vous pouvez configurer le périphérique Maple Systèmes pour utiliser l'une des vitesses supportées (voir le volet JMRI préférences pour les connexions Maple), et utiliser le "Protocole simple ASCII". Plus d'informations sur le protocole et la manière de configurer le périphérique Maple Systems est sur la feuille de données .

JMRI communique avec le périphérique Maple Systems en lisant et en écrivant dans des empacements "discret mémoire". Lieux, autrement connu comme registres "LB".

LB registres 1 à 48 sont disponibles pour JMRI comme entrée «Capteurs» 1 à 48.

LB registre 101 à 148 sont disponibles pour JMRI comme sortie "Aiguillages" 1 à 48.

Le nombre nœud Maple Systems (programmé dans "Paramètre 5", la "station HMI ID #») peut être de 1 à 99. Il apparait comme le chiffre des milliers et des noms de Capteur et Aiguillages JMRI. Par exemple, KS3001 est le 1er capteur sur le noeud 3, et serait donc LB 1 dans ce Système d'unité Maple . KT3001 est le 1er aiguillage de ce nœud, et serait donc LB 101 dans cette unité.

Il est très probable que nous allons retravailler et améliorer l'adressage bientôt.

Vous devez configurer un nœud en JMRI avec le bouton "configurer les nœuds» sur la fenêtre des préférences. Sur la nouvelle fenêtre qui apparaît, entrez le numéro de nœud (par exemple 1 ou 2, etc) dans le champs "UA", et cliquez sur "Ajouter". Laisser tout le reste seul; Le support Maple a été créé à partir du support JMRI C/MRI, et en 2.3.7 JMRI la migration n'est pas complète.
