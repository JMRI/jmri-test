JMRI: Détails Régulateurs RailDrive

Cette page est en construction. Le support RailDriver JMRI a d'abord été disponible en JMRI 2.1.7 . Il évolue rapidement, alors revenez ici ou sur le jmriusers mailing liste pour des changements.

Alors que nous développons le support RailDriver, il est principalement mis en œuvre via un script RailDriverThrottle.py . Ceci permet aux utilisateurs de l'éditer facilement pour le personnaliser suivant leurs désirs tandis que nous trions ce qui est généralement nécessaire.

Structure

Parce jinput et le RailDriver fournissent des noms pour les différents composants sur Windows, MacOS X et Linux, le script est organisé autour du nombre (index) de la composante, et non pas de son nom. Les trois plates-formes fournissent les composants dans le même ordre.
