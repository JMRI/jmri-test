JMRI: Construction avec XCode

Introduction

Les versions récentes de Mac OS X sont livrés avec Xcode , un système de logiciel de développement élaboré pour le Macintosh d'Apple Computer. Xcode peut être utilisé pour travailler avec JMRI.

Les instructions ci-dessous supposent une extraction récente de JMRI, avec une version récente de Xcode.

Procédure pour démarrer JMRI avec Xcode

  1. Démarrer Xcode, , rejeter n'importe quelle fenêtre qui apparaît, décochez la case et le retour à cette fenêtre, etc

  2. Sélectionnez Window> Organizer . (Note: Xcode utilise maintenant le fenêtre Organizer pour Java, au lieu de la fenêtre principale Xcode utilisée dans les premières versions de Xcode.)

  3. En bas à gauche de la fenêtre de l'organiser, dans le " + " apparaît, sélectionnez «Ajouter un Dossier existant ...

  4. Accédez au dossier " java "dans le pack JMRI, et cliquez sur Ouvrir .

  5. Sélectionnez " java " dans la fenêtre Organizer. Cliquez sur l'icône Clean en haut de la fenêtre Organizer. Lorsque cela est terminé, maintenez la touche icône Build et sélectionnez l'icône ant'compile . Cela va compiler toutes les sources JMRI.

  6. Sélectionner Build ant 'jar' va construire un fichier jmri.jar. Le jar est placé dans votre dossier de commande.

  7. Pour afficher ou modifier un module de source existante, ouvrez le dossier java en haut à gauche dans la fenêtre Organizer (cliquez sur le triangle à gauche), puis ouvrez le dossier src , puis le dossier apps ou jmri , etc pour trouver le fichier qui vous intéresse

  8. Pour créer ant 'panelpro' entrée dans la liste de construction:

    1. Sélectionnez Edit Actions ..." dans la liste de construction.

    2. Sélectionnez ant 'compile' dans la liste à gauche, et copiez la commande qui s'affiche dans le boîte sur la droite.

    3. Dans le + déroulant en bas à gauche, sélectionnez >New Shell Script

    4. changement de " Shell Script " pour ant 'panelpro' " dans la liste sur la gauche.

    5. Collez le texte que vous avez copié dans le b dans la case de droite, et le changer " compilation " en " panelpro ".

    6. Cliquez OK .
