Provides a "configuration wizard" application for the JMRI suite of programs.

The main routine for this application is in the {@link apps.wizard.JmriWizard} class, which extensively uses support from the {@link apps} package.

Maintenance issues

Currently, this package is manually updated to know about the other applications and their configuration files.

The structure of window classes is:

  • ApplicationPane - select application, hence resulting config file
  • SelSystemPane - select system (hardware) type
  • (system).Select
    • easydcc.Select - select TCP/IP or serial
      • Serial:
        • SelectSerialPort - select from available serial ports
      • TCP/IP:
        • SelectTcpInfo - enter TCP/IP info
    • nce.Select - select TCP/IP or serial
  • () - confirm & finalize
