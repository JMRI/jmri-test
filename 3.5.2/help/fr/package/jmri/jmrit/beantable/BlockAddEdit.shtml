Ajouter/Modifier les entrées de Table Canton

Entrez le nom système pour le Canton que vous voulez ajouter. Par exemple, un nom valide comme nom système de Canton pour un réseau pourrait être LocoNet LB456. Un autre exemple, un nom système interne valide pour un capteur pourrait être IB123. Le nom utilisateur facultatif peut être quelque chose que vous voulez pour décrire le Canton. Voir la page d'aide " Noms " l pour plus d'informations.
