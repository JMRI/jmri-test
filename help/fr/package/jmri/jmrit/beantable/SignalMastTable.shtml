La Table Mât de Signal

Pour plus d'informations sur les Mâts de Signaux et les aspect de la signalisation, voir la page introduction .

Pour définir un nouveau mât de signal , cliquez sur le bouton" Ajouter .." . Il vous sera demandé toutes les informations de configuration nécessaires.

Utilisez l'option Stocker dans le menu Fichier pour enregistrer les informations de signalisation sur le disque. Ces informations sont enregistrées avec la configuration dans un fichier XML, avec l'installation du panneau de commande et des chosess similaires. Pour stocker vos panneaux dans le même fichier (recommandé), sélectionnez Fichier> Stocker > Stocker Configuration et les panneaux dans Fichier ... ou sélectionnez Panneaux... dans la fenêtre principale JMRI.

Pour plus d'informations sur la création et l'affichage des panneaux, y compris la manière de montrer des signaux sur vos panneaux, s'il vous plaît voir les pages Panneaux.
