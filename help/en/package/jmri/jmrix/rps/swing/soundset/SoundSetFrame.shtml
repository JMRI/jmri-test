JMRI: RPS Sound Speed Control


This window lets you monitor and control the speed of sound used by the RPS system.

It displays the current value near the top. Depending on how the system is operating, this might change occasionally.

Below that are some controls.

First is a section where you can enter a new value.

If you have a transmitter in a fixed position, you can also use that to automatically measure and set the speed of sound.
