Digitrax Firmware Downloader

Some Digitrax products can have new firmware (internal programs) downloaded to them. The firmware updates are distributed as ".dmf" (Digitrax Mangled Firmware) files. They contain both the code itself, and various managment information to make sure that it can only be downloaded into the hardware it expects.

Note that this is not the same as the Sound Download Tool, which is used with a PR2 or PR3 to load new sound projects into a Digitrax sound decoder.

To use this tool:

  • Click the select button, and select the .dmf file you want to download.
  • Click Read File to read the file into the program and verify its contents
  • Only if instructed to by Digitrax, change the various values in the control fields (this is usuall not needed)
  • Click Download to load the firmware. This will take a little while, so there's progress bar below the buttons.
  • If desired, click Verify to double-check the contents. This also takes a while.
