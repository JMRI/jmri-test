Configure Digitrax PM4

The JMRI PM4 programming tool lets you configure the internal options of a PM4 directly from your computer.

To open this tool from the JMRI start page select the Loconet tab then find the tool on the drop down list.

To use it, you put the address of a PM4 board in the text field and click "Read from PM4". The tool will read back the current state of the various options, and set the check boxes to correspond to their current values.

You can then, if you wish, change the checkboxes and click "Write to DS64" to make your changes permanent.

Limitations

Because of the way the PM4 board works, this tool can't change the basic address of the unit.
