Set NCE USB Cab Id

This tool will show which address a NCE-USB is using and allow the user to set the Cab Bus Id for the interface.

The display will show what the cab id of the first NCE-USB interface it finds. It will not show all of them if there were multiple NCE-USB interfaces connected to a system. To see all of them, use the Cab Monitor for a listing.

This command uses the memory commands to find which cabs are the NCE-USB type. So currently that only works for the PowerHouse command station or an updated NCE-USB (version 7.* or better) and system running the 1.65 Firmware or better.

Enter the new number in the in the 'New Cab Id' field and hit the 'Set' button.
