Renumber Grapevine Node

This is the help/jmri/jmrix/grapevine/nodetable/RenumberFrame help page

This tool lets you change the address of a Grapevine node.

The node remembers the change, even through power off/on cycles, until you change it again.

Enter the original node number, the new node number, and press "Change".

You might want to open the Node Table and click "Check" to see if the renumbering has worked.
