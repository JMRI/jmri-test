JMRI Throttle

This is the quick reference page for the JMRI Throttle tool.

For more information, please go to the general throttle page.

Hints

The "Program" button in the lower right opens an ops-mode (programming on the main) DecoderPro programmer for the selected locomotive. It's only active when a roster entry has been selected on the throttle.

Changing the Window Title

If you select "Frame Properties" from the Edit menu on the throttle window, you can change the title on top of your throttle window. There are four choices you can select from:
  • Address - Automatically display the address of whatever locomotive is selected. This is the default.
  • Text - Display the text that you enter in the field at the top of this window
  • Text Address - Display the text you entered, followed by the selected address
  • Address Text - Display the selected address, followed by the text you entered
Note that if you enter text, it won't be visible unless one of the last three modes is selected.
