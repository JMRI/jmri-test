JMRI Throttle

This is the quick reference page for the JMRI Throttle tool.

For more information, please go to the general throttle page.

Keyboard Shortcuts

You can operate the Throttle using your keyboard. Keys include:
Numeric Keypad 0 - 9
Functions 0 to 9
Numeric Keypad .
Function 10
F11 - F24
Function 11 to Function 24
Numeric Keypad +
Increase Speed
Numeric Keypad -
Decrease Speed
Left Arrow
Increase Speed
Right Arrow
Decrease Speed
Page Up
Big Increase Speed
Page Down
Big Decrease Speed
Up Arrow
Reverse
Down Arrow
Forward
Numeric Pad /
Stop (estop)
Numeric Pad *
Idle (speed 0)

Hints

The "Program" button in the lower left opens an ops-mode (programming on the main) DecoderPro programmer for the selected locomotive. It's only active when a roster entry has been selected on the throttle.
