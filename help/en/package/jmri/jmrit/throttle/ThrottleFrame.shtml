JMRI Throttle

This is the quick reference page for the JMRI Throttle tool.

For more information, please go to the general throttle page.

Hints

The "Program" button in the lower right opens an ops-mode (programming on the main) DecoderPro programmer for the selected locomotive. It's only active when a roster entry has been selected on the throttle.

Changing the Window Title

If you select "Frame Properties" from the Edit menu on the throttle window, you can change the title on top of your throttle window. There are four choices you can select from:
  • Address - Automatically display the address of whatever locomotive is selected. This is the default.
  • Text - Display the text that you enter in the field at the top of this window
  • Text Address - Display the text you entered, followed by the selected address
  • Address Text - Display the selected address, followed by the text you entered
Note that if you enter text, it won't be visible unless one of the last three modes is selected.

Preferences

The Throttle Preferences, accessed from the main menu, control how the throttle window looks:
Use extended throttle
Enable/disable the whole bunch of features I've added, when disabled, throttle windows have the same behaviour than before.
Use roster image as background
If a roster image is defined in the roster entry, it will be used as background in the throttle pane.
Resize window to image size
Will force throttle pane to resize to actual roster image size (overriden by throttle window size when backup file exist)
Set controls transparent
Allow AddresPanel, ControlPanel and FunctionPanel to be transparent, quite nice with background image. Due to a bug in UI implementation, might not work in Linux, but is very nice on OSW, not sure on Windows.
Search roster info when address entered
When an address is entered in the AddressPanel instead of a roster entry being selected, will search in the roster list if there is one matching this address and then automotically update the throttle frame.
Automatic load of throttle window preferences when linked to roster
When a roster entry is selected, if there is a matching throttle file (based on roster id) automatically load these preferences (windows position and size, and icon status; I've created a reduced throttle backup file as function buttons label and background image are already in the roster entry). This throttle file is created when choosing "Export customization to roster", automatically saved as "roster ID.xml" in throttle folder in JMRI preferences folder.
Hide undefined roster function buttons
When using a roster entry, it will hide function buttons that have no label defined. Also hide the alternatives buttons if no function text defined over 15.
Apply
Apply curent preferences, a new throttle window shall be opened to enjoy.
Save
Apply, Save to xml file and close preferences window.
Cancel
Close the preferences window.
