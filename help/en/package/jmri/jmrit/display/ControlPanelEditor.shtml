The Control Panel Editor

Control Panel Editor is simply an alternative view and controller of exactly the same content data as Panel Editor. You may change the view and control of your panel whenever and as often as you please. Storage and loading can be done from either editor. The editor used to store the panel will be the editor used at the next load.

Note: Control Panel Editor was developed and tested on Windows only. The developer would very much like to have this application work on other operating systems (e.g. Mac and Linux), but is unable to test on these environments. Any comments would be deeply appreciated.

Changing the View between Editors

In either editor when in edit mode the last item in the File menu is Change view to "...". ("..." being the other editor). Selecting this item switches the view from one editor mode to the other editor mode.

Differences in the View between Editors

Panel Editor has a separate window for its editing functions. Control Panel Editor does not. Instead three new menus are introduced to the panel: Options, Zoom, AddItems. When operating mode (non-edit) is selected these three menus disappear.

Control Panel Editor Menus

File Menu

The File menu changes it items between edit mode and operating mode. In edit mode the items are exactly the same as the items in the File menu of the Panel Editor's edit window with one additional item, Rename Panel. In operating mode Control Panel Editor's File menu has only one item, Open Editor.

Options Menu

The items in this menu are similar to the checkboxes in Panel Editor's edit window with a few additional items and features.

The first item is Close Editor. This changes Control Panel Editor's mode from "edit" to "operating".

The next item, Override individual Position & Control settings, enables a feature not found in Panel Editor. When this item is selected, the next two items are enabled. Selecting these item allow you to reposition all items in the panel or to have all items with the capability of controlling the device they represent to control the device or not.

Unchecking the Override individual Position & Control settings item will return the individual item settings to their previous settings. This feature is not available in Panel Editor, where checking these options will permanently set all panel item to those set by the Panel Editor.

Setting scroll bar preferences is also found in this menu.

Zoom Menu

The Zoom(percentage)... item will enlarge or reduce the panel size.

The Zoom to Fit item will enlarge or reduce the display of all panel items in the panel to fit within the panel window without the use of scroll bars. This can be useful if your panel design happens to be slightly larger that what you can fit on the screen. Rather than using scroll bars to see all items on the panel, "Zoom to Fit" will reduce the display to fit the panel window. Conversely, it will expand the display to fit the panel window. If you resize the panel window, then you should repeat "Zoom to Fit" to resize the display to fit the new panel window dimensions.

AddItems Menu

This menu will open a tabbed window which allows you to add items to the panel. This tabbed window, Item Palette, extends the functions found in the drop down ComboBox of the Panel Editor. The methods for adding items done by drag and drop. Details may be found at !!!ItemPaletteHelp - to do!!!.

Mouse operation in Control Panel Editor

Control Panel Editor does not implement the rollover highlighting found in Panel Editor.

Another difference is selection is done with a simple click. Right click (meta-click on Mac or Linux) brings up the popup menu for the item.

Dragging can be donw with either a simple click or a meta-click.

Multiple Selection

As with Panel Editor, a selection group can be made using the "selection fence" - a dotted red rectangle that will select all the items completely within the fence. If the Shift key is held down while creating a selection fence, then all items that intersect the dotted red rectangle are selected. A useful feature when items are located at the top and left borders of the panel.

Another shared feature is adding or deleting items from a selection group by holding the Control key dowm whhen slecting the item.

Copy in Control Panel Editor

Copying an item or selection group is done by bringing up the popup for one of the selected items. When the Copy popup menu item is clicked, a Copy of the selection group is made on top of the original selections. Drag this copy to whereever you wish to place the copy. The original selection is now deselected and remains at its original postion. The copy procedure can be repeated on the newly positioned copy to continue to make more copies.

Item Popups in Control Panel Editor

Right click (meta-click on Mac or Linux) brings up the popup menu for the item.

Edit Text Attributes

The Edit Text Attributes popup item opens a dialog to set any of the item's text attributes.

Multiple Selection Editing

When a popup menu item is invoked from an item that is a member of a selection group, all of the edited values that were made in the popup are applied to all members of the selection group if that value is relevant to the member. For example, if the font size is changed on one of the items in the selection group then the font size is changed in all items in the group. If say, a fast clock were included in this group, then no change would be made to the clock.
