The Item Palette

The Item Palette is used to place icons and text onto Control Panel Editor panels. The palette is a tabbed window with tabs for each item type. The common method all items use to add an item to a panel is "Drag and Drop". Each tab has a bordered panel labeled Drag to Panel. Drag the item inside the border to the panel. It some cases, you may have to make a selection before dragging - e.g. select a row from a table or enter text into a field and press the enter key.

Iconic Items

For those items using icons, multiple sets of icons are offered by the Item Palette. In each of these cases several icon sets are provided by default. Radio buttons are used to select an icon set. The Show Icons button displays all the icons in the set. The Edit Icons button opens a dialog where the icons may be changed by dragging new icons from the icon catalog.

New Icon Sets

Icon sets may be added or deleted. There is no limit on the number of additional icon sets that can be added to an item tab. When the Item Palette is customized with other icon sets, the Item Palette is stored in the CatalogTrees.xml file in the preferences directory.<\p>

The Edit Icons dialog is used to delete the icon set or to add an additional icon set. When the Add New Icon Set button is pressed, the currently selected icon set is saved and an icon set with "unknown" icons is displayed. Change these icons by dragging icons from the icon catalog and be sure to name the new icon set in the text field at the top of the dialog. Finally, press the Create Icon Set button and the dialog is closed with the new icon set added to radio button list.

Tabled Items

The items associated with a table have an Add New Table Item button.

Item Palette Tabs

  • Turnout Both left and right turnout icon sets. Select a turnout from the table.
  • Sensor Select a sensor from the table.
  • SignalHead When a signal head is slected in the table, only the icons for the valid appearances of the signal are shown. When no row is selected all possible appearance icons are shown.
  • SignalMast A signal mast must be selected from the table to display the icon it used for its aspects.
  • Memory Select a memory location from the table. The three memory types each have a "Drag to Panel". Input and Spinner types can only be dragged from the edges of their text boxes.
  • Reporter Select a reporter from the table.
  • Light Select a light from the table.
  • Multisensor The default icon set has three lever positions. The table displays the Sensor Table. Three rows must be slected from the table to place the default icon in a panel. The table selections are labeled "first", "second", etc. and these labels correspond to the labels shown when the full icon set is displayed. When editing an icon set, level positions can be added and deleted.
  • Icon The bordered icons displayed (labeled with their names) can be dragged directly to a panel. Icons can also be dragged directly from the Icon Catalog.
  • Background Background icons are placed in a panel just as regular icons. However, they are place at the lowest display level and cannot be selected with a simple mouse click. Use "Control Click" to select a background icon. A custom background color can be chosen with the Background Color button.
  • Text The Text tab displays the same dialog as the "Edit Text Attributtes.." popup menu item with the addition of a text field to enter the text. Font sizes styles, etc and border attributes can be chosen and a color palette can supply custom colors. The "Enter key" must be pressed to display the text with its attributes in the Preview Panel. Drag the item in the Preview Panel to the panel.
  • RPSReporter Drag the "Drag to Panel" icon to the panel and setup the RPSReporter from its popop menu.
  • FastClock Drag the clock to the panel.
