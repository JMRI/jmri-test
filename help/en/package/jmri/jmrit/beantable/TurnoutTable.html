The Turnout Table

This table contains a list of all turnouts that exist for this panel.


Turnout Table Buttons and Checkboxes

  • User Name
    You can directly enter or change the "User Name" for an existing turnout.
  • Cmd
    Pushing the "Cmd" command button will change the state of an existing turnout.
  • Delete
    Pushing the "Delete" button will remove the turnout from the table. To permanently delete the turnout you must save the panel.
  • Inverted
    When checked, the turnout command and status to and from the layout is inverted. For example, an inverted turnout means that when JMRI sends a "CLOSE" turnout command, it becomes a "THROW" to the layout. Turnouts that can not be inverted have grayed out boxes and can not be checked.
  • Locked
    When checked, the turnout is locked. This means that turnouts that can be locked by JMRI will not respond to commands from cabs to change. Turnouts that can not be locked have grayed out boxes and can not be checked.
  • Feedback
    This shows the known state of a turnout.
  • Mode
    This allows you to change how the turnout will operate.
  • Sensor 1
    Shows the system name of the optional sensor that provides the feedback for a turnout. You can also directly enter or change the system name of Sensor 1 for a turnout.
  • Sensor 2
    Shows the system name of the optional sensor that provides the feedback for a turnout. You can also directly enter or change the system name of Sensor 2 for a turnout.
  • Add
    Pressing this button allows you to add a new turnout.
  • Show feedback information
    When checked, the optional feedback columns are shown.

NCE Turnout lock feature

The turnout lock feature allows you to selectively disable accessory (turnouts) commands from cabs. By locking a turnout, the JMRI program will monitor the turnout's accessory address and if it detects a change command from a cab, will cancel the command immediately by sending out the opposite command to the turnout.

To use this feature you must select MONITORING as the feedback mode for the turnout and the NCE command station must have the 2007 or newer EPROM.

To help you select a set of turnouts to lock, a route can be locked. All turnouts that are being monitored by the JMRI program and are included in the route will be locked.
