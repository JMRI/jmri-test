Adding Signal Masts

Signal Masts are built based upon different signalling systems, and the masts supported within those system.

In the add window, you first need to select the prototypical signalling system that the mast is to be based upon, followed by the mast within that system.

The mast can then be built in three different ways depending upon the hardware being used.

  • SignalHeads
    • Individual signals heads are used to represent each signal head on the mast. The signal head will then deal with handling the changing of it aspect depending upon the state of the signal mast. If each signal head that has been configured on the mast, a drop down box will appear for you to select the signal head that will control the head on that mast.
  • Turnout Driven
    • This is for specific hardware that can control the whole appearance of a Signal Mast based upon the setting of a turnout. Depending upon the hardware used will depend upon what needs to be set here.
  • Virtual
    • This mast does not directly control any hardware relating to a signal mast. It use is ideal for where a physical mast would not be present on the railroad, such as an End Bumper/Buffer Stop, or to mark the end of a signalling section of track and the beginning of dark territory.
