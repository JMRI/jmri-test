Add/Edit Special Action

The Add/Edit Action Window

Special Actions are things that can be initiated automatically as a train running in automatic mode traverses the layout. The Add/Edit Action window (see below for "Add" version) provides for creating new Special Actions and for editing current Special Actions. The "Edit" version of this window is the same as the one shown below, except the Create New Action button is replaced by an Update Action button and the selections and data are filled in to current values of the Special Action being edited.

A Special Action is defined by specifying What action is requested and When that action should be initiated.

To specify the When part, select an option in the When popup menu, and fill in any data that might be required. Data item fields or menus will appear when the item needing them is selected. Each data item is supplied with a text description that may be summoned by hovering the cursor over the field. Each When option is described in more detail in the View Special Actions help file.

An optional delay, specified in milliseconds, is available for each When option. No delay is the default. If an optional delay time is not entered (or a delay time of zero is entered), the action will be initiated immediately after the when event occurs. If a delay time is specified, the program will wait for the specified time after the when event occurs before initiating the action.

To specify What action is requested, select an option in the What popup menu, and fill in any data that might be required. Data item fields will appear when the item needing them is selected. Each data item is supplied with a text description that may be summoned by hovering the cursor over the field. Each What action is described in more detail in the View Special Actions help file.

When the Special Action is completely defined, click the Create New Action button (or the Update Action button if editing). Any required data will be validated. If a problem is found, a message will appear, and the Add/Edit Action window will remain to allow correction of the problem. If everything checks out, the Add/Edit Action window will disappear and the new (or edited) Special Action will appear in the View Special Actions window.

Click the Cancel button to dismiss the Add/Edit Action window without creating a new Special Action or completing the edit of an existing Special Action. Closing the Add/Edit Action window is equivalent to clicking the Cancel button.
