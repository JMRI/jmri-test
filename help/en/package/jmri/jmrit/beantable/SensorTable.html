The Sensor Table

This table contains a list of all sensors that exist for this panel.


Turnout Sensor Buttons and Checkboxes

  • User Name
    You can directly enter or change the "User Name" for an existing sensor.
  • State
    Pushing the "State" command button will change the state of an existing sensor.
  • Delete
    Pushing the "Delete" button will remove the sensor from the table. To permanently delete the sensor you must save the panel.
  • Inverted
    When checked, the sensor command and status is inverted. For example, an inverted sensor means that when the layout sends an "active" sensor command, it becomes "inactive" to JMRI.
  • Add
    Pressing this button allows you to add a new sensor.
