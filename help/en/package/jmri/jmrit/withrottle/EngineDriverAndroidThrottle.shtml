JMRI: Engine Driver Throttle for Android

Engine Driver version 0.3 is now available. This version includes numerous cosmetic and error-handling improvements, and allows the user to set function labels (or copy them from JMRI roster if available).

This application can connect to a JMRI WiThrottle server running on a PC, and in turn control your locomotives. Speed, direction, and up to 16 DCC functions are supported. 

Prerequisites:

Verify your Android can connect to your PC network via WiFi and obtain an address.
Make sure you are running one of the latest versions of JMRI (2.8 works, 2.9.6 works better) on your PC, connected to the layout.  Start up JMRI and verify you can use the included "Throttle" app to control a loco.

Installation:

Before your Android device will allow you to install this program, you will need to permit the installation of non-Market applications. This is under Settings->Applications->Unknown sources. Then use your phone to download this file (or download and copy from your PC).  Click on the file to begin installation.  Follow the prompts as usual.  (Note: if you have v0.1 or v0.2 installed, you will need to uninstall it manually.)

Operation:

Start the WiThrottle function of JMRI, located in DecoderPro under Tools, Throttles.  (Or under Debug in older versions of JMRI).

On your phone, start the Engine Driver app and wait for your PC's address to show up in "Discovered Services".
Click on the discovered address, (or you can enter the address and port of your WiThrottle server directly).

On the next screen, enter the loco address (verify short or long), and press "Connect".

Operate your loco using the slider for speed, and the reverse, stop and forward buttons to control direction.
To release the loco, just press the Back button. This will also stop your loco (can be overridden in preferences).

Configuration:

Under Preferences, set the name you'd like to call your throttle.  (Must be unique if multiple EngineDrivers in use)
Also, you can turn off the "stop on release" feature if desired.
Use the Function Buttons menu option to set the labels and function numbers to display. If you have set labels in a roster entry in JMRI, these labels can be copied from the dialog (access the menu after selecting a loco).  Remove buttons by blanking out the labels.
