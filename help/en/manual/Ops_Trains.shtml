The Trains window opens which allows you to build trains to run on your layout.

Menu Bar

 

  • Tools
    • Print -- opens printer dialog for your default printer.
    • Preview -- opens printer dialog for your default printer. OK then preview is displayed.
    • Print Options -- opens Print Options dialog that lets you set up your manifest printing.
    • Modify Trains -- opens Modify Trains dialog to allow you to modify a train.
  • Windows -- standard DecoderPro® format.
  • Help -- standard DecoderPro® format.

Select a Sort by Name (default)

At the bottom of the window is a series of buttons, at this time press to open the Add Train window.
