Main Window Overview

Back to Index

When you start Decoder Pro, the DecoderPro window is displayed. From this window you can access all the functions of DecoderPro.

Menu Bar

Provides access to the menus that allow you to access all the features of DecoderPro

Service Mode (Programming Track) Programmer button

Provides access to the decoder programming capability in Service Mode Programmer.

Operations Mode (Main Track) Programmer button

Provides access to the Operations Mode Programmer.

Help button

Opens the Help Window

Quit button

Exists the program, with confirmation dialog.

Additional Buttons (Optional)

The buttons shown at the bottom of the window are not JMRI defaults, but instead are setup in your preference to open each time you start DecoderPro®. The following are samples that could be set in your preferences. (Shown are for a Digitrax LocoNet system)

New Throttle.. button

Opens a throttle window provides a tool to control your trains, you may open more than one throttle and position on the computer screen

LocoNet monitor button

Opens the LocoNet Traffic window that allows you to monitor the commands and responses on the LocoNet.

Power Control button

Opens Power Control window, which allows you to turn track power on or off during operations.
