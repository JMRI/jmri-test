Tool Bar

Opens the DecoderPro3 help file
Opens the new locomotive window to program a decoder
Attempt to Identify the decoder in the locomotive
Allows you to select the Programming Mode

Roster Table

Contains a list of decoders in your Roster. The entries can be highlighted with the mouse, and deleted, edited or have other actions performed on them.

Decoder Information Area

Displays the information about the current decoder selected.

You may select the progamming mode:
Service Mode Programmer
Operations Mode Programmer
Edit Only
Opens the Basic Programmer for the selected decoder in the Roster List
Opens the Comprehensive Programmer for the selected decoder in the Roster List
Opens the Functions labels window and allows you to label throttle function buttons
Opens the Roster Media window and allows you to assign an image to the roster entry
Opens a throttle tool for the selected loco in the Roster List.


Status bar

Displays message, so that you can determine the state of DecoderPro3 during operation.
