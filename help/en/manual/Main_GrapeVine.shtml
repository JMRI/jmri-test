Communications Monitor

Opens the Communications Monitor Window and automatically displays the information that is being sent and received on the communications link between your computer and your DCC system.

Send Command

Opens Send Grapevine serial command prompt. Type Command and click the Send button to send the command via serial Bus. Type node Address and click on Query Node button to query the Grapevine serial bus node.

Configrue Nodes

Opens Configure Nodes window, which provides the tools for setting the Grapevine Serial Bus nodes.

Node Table

Opens Grapevine Nodes window which displays a table of the nodes.
