JMRI: Grapevine System Names

Individual inputs (Sensors) and outputs (Turnouts and Lights) in JMRI are accessed via names. For information on the general scheme, please see the naming page in the technical documentation.

For Grapevine systems, all names start with a system letter of "G". After that, there's a type letter, then a node number and a pin number.

Each Grapevine node has a number, from 1 to 127.

The pin number, from 001 up, defines the specific input or output connection on the node. Note that a pin number is always three digits.

For example, the 1st Grapevine Sensor input pin on node 23 would be named GS23001. The 5th Turnout on node 2 would be GT2005.

If it's helpful, you can think of the node number as being the "thousands" part of the combined number, and the pin number as the lower three digits.

Inputs

Grapevine nodes have a number of possible inputs:
Parallel Inputs
There are eight three-pin connectors on a Grapevine node that can be used as general inputs. These Sensors are referenced as 001 to 008.

In addition, there are eight additional pins in the 14-pin connector that can be referenced as Sensors 009 to 016.

Serial Occupancy Sensor Cards
Up to three serial occupancy sensor cards can be attached to each node. Each one senses occupancy on up to eight track segments.
