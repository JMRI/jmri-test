JMRI: USS CTC Tools

JMRI contains some tools to help in the automation of USS CTC panels. These can be used for either screen-based panels, or to drive a physical panel with real levers and lights.
Turnout Lever
Controls a turnout from a lever on the panel, along with the two indicator lights.
Signal Lever
Controls a set of signals from a lever on the panel, along with the three indicator lights.
Code Button
Implements the code button and code lamp.
OS Indicator
Drives the occupancy indicator lamps on the panel
Follower
