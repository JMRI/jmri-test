JMRI: Signal Masts

Background information on JMRI Signal Masts

The JMRI "SignalMast" objects allow you to display prototypical aspects on signals. For background information, see the Aspect Signaling page.

Signal Masts are configured through the Signal Mast Table You add new ones with the "Add..." button on the table, which takes you to the "Add Signal Mast" window.

From the table, you can set individual signal masts to any aspect that they're able to display by clicking in the "Aspect" column.
