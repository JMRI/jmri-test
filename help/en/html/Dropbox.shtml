JMRI: Sharing Files with Dropbox

Dropbox is a free service that lets you share files between multiple computers across a network. Once you've signed up with Dropbox, you can configure JMRI to use it to share rosters, layout configuration files and other information between multiple computers.

To set this up:

  • Create a JMRI folder in Dropbox on the first computer.
  • (If you have already created items (roster entries, panels etc.) go to Help->Locations and press Open User Files Location (and also Roster Location, only if it differs from User Files Location). You will need to move your items from there later.)
  • Preferences->Roster->Roster->Roster Location->Reset. Do not enter a Roster Location. That way it will follow User Files Location.
  • Preferences->Locations->User Files Location and set it to the JMRI folder you created in Dropbox.
  • Restart JMRI and use Help -> Locations to make sure everything is changed correctly.
  • (If you have already created any of the items listed at this link, move them from the old location to the new User Files Location: see this page. )

The only steps you need to do on subsequent machines are:

  • Preferences->Roster->Roster->Roster Location->Reset. Do not enter a Roster Location. That way it will follow User Files Location.
  • Preferences->Locations->User Files Location and set it to the JMRI folder you created in Dropbox.
