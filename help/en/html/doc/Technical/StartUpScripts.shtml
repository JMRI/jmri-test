JMRI: Startup Scripts

This page describes the scripts used to start up JMRI

Common Aspects

The basic things needed are:
  • Form and provide a classpath
  • Provide system options and workarounds
    -noverify
    -Dsun.java2d.noddraw
  • Configure the security manager
  • Configure RMI
  • Finally, invoke the appropriate main entry point.
