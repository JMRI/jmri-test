Guide d'Installation JMRI: Centrale Twin Fleischmann via Port Série

DecoderPro peut communiquer avec une Centrale Twin Fleischmann via le port série intégré et le protocole LocoNet.

Notez que cette fonctionnalité a été ajoutée à JMRI en test sur la version 1.5.2, et n'est pas présente dans la Version précédente 1.4 de production. Si vous utilisez une version avant la version 1.5.2 JMRI, S'il vous plaît aller à la page de téléchargement pour télécharger la dernière version.

La Centrale Twin a besoin d'avoir ce logiciel mis à jour à la version 1.1 ou ultérieure.

Configurer les options de votre Centrale Twin à:

  • SO 2 valeur 4
  • SO 5 valeur 1
  • SO 6 valeur 255
  • Baudrate 19200
Ensuite, démarrez le programme JMRI et le configurer pour:
  • "LocoNet Intellibox Serial Port" protocole
  • 19200 bauds
  • Sélectionnez matériel Handshake
  • Sélectionnez la centrale de commande DCS100
