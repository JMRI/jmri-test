JMRI: Programmation Spécialités Produits DCC

DCC Specialities fait plusieurs contrôleurs puissants pour les chemins de fer modèle. Leurs deux contrôleurs de moteur switch sont appelés les Hare et Wabbit. Ils font aussi un détecteur intelligent appelé le bloc de Block Watcher, et les puissants PSX etPSX-AR contrôleurs de puissance.

Ceux-ci peuvent faire beaucoup de choses, mais ils peuventt exiger du travail pour les configurer.

Brett Scott et Denny Whitaker ont écrit des scripts JMRI pour aider à les programmater, et ontfourni une belle writeup , (lien cassé le 17/01/2012) sur la façon de les utiliser. Le writeup couvre également comment elles documentent leurs réseaux, et discute les avantages et inconvénients de plusieurs manières de connecter le Hare et la Wabbit.

vous pouvez télécharger leurs scripts depuis le site web JMRI, et il est aussi disponible dans des exemples de répertoires de scripts dans le Téléchargements JMRI .
