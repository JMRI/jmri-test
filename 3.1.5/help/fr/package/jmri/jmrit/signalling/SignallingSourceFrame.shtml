JMRI: Cadre Logique du Signal de Source

De cette fenêtre, vous pouvez voir touts les logiques de signal qui affectent l'aspect du mât de signal, tout en étant capable d'éditer chaque logique.
Si l'éditeur de réseau est utilisé, il est possible de découvrir automatiquement toutes les logiques signalmast basées sur le panneau.
