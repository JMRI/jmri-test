Partage Des fichiers DecoderPro à Travers Un Réseau

Une question commune :
J'ai un ordinateur à mon plan de travail, où je programme mes machines, et un autre à l'emplacement de mon circuit. Ils ne sont pas portables, mais tous les PC de chez moi sont reliés entre eux (les clients Windows XP et Windows 2000 Server). Je voudrais pouvoir stocker le fichier et tout ce qui est nécessaire dans le serveur Windows 2000 et être en mesure d'exécuter des copies de Decoder Pro à partir des deux Windows XP PC. Est-ce possible?
Oui!

Configurer les machines Windows il y a donc un répertoire partagé, et puis vous pouvez définir les "préférences avancées" de JMRI pour mettre l'information à l'intérieur de ce fichier.

  1. Mise en place d'un répertoire partagé.
  2. Sélectionnez le fichier avec lequel vous voulez commencer (car il peut se séparer de ceux déjà présent sur des machines différentes). Sur cette machine, trouver le fichier "roster.xml" et le répertoire "Roster" dans le répertoire des préférences..

    Pour plus d'informations, s'il vous plaît voir les pages sur DecoderPro the Fichiers Liste et Emplacements des fichiers.

    .Copier, _Ne pas déplacer_, le fichier roster.xml et le fichier dossier dans le dossier partagé. Vous devez copier le contenu_complet_du fichier du répertoire !

  3. Sur chaque machine qui utilise le fichier partagé, démarrer DecoderPro, sélectionnez Préférences dans le menu Edition, puis cliquez sur "Afficher les Préférences avancées". La fenêtre sera plus grande. Appuyez sur "Set" sous la rubrique «Liste:" et suivez les instructions.
