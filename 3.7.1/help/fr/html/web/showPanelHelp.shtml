JMRI: application web showPanel

Contexte

JMRI FrameServlet est un outil utile pour étendre JMRI aux navigateurs en saisissant périodiquement une copie d'écran d'une fenêtre demandée (fonctionnant sur un serveur JMRI), l'afficher dans le navigateur, et transmission de tout clics de souris clique en retour à la fenêtre JMRI. Malheureusement, il y a quelques inconvénients à cette approche (réponse différée, la charge importante du serveur, pas de mise en valeur des éléments actifs, souris "partagée", etc.).

Avec l'ajout de PanelServlet JMRI, il est devenu possible pour une application cliente de récupérer les détails complets d'un panneau, il suffit de recréer le panneau dans le client. La XMLIOServlet fournit la mise à jour des éléments JMRI, et pour le suivi des modifications apportées à ces éléments.

Nouvelle application web

showPanel.html est une application web moderne qui dessine les panneaux JMRI actifs sur un écran du navigateur. Il récupère les définitions de panneaux de JMRI via le PanelServlet et panneau construit côté client, y compris les fonctions de clic. Il envoie des changements et à l'écoute des changements touchant des éléments du panneau en utilisant le serveur xmlio.

Les Liens vers tous les panneaux disponibles sont affichés sur la page d'accueil du serveur Web JMRI (vous pourriez avoir besoin de le reconstruire dans Prefs pour voir les liens)

l'URL de base est "/web/showPanel.html". En l'absence de paramètres, la page liste les liens vers les panneaux disponibles, ou vous pouvez passer un paramètrem du format "? name = [type]/[panelname]" pour ouvrir directement le panneau spécifié. Par exemple: http://192.168.0.7:12080/web/showPanel.html?name=Layout/MyPanel "

Cette application nécessite un navigateur moderne "HTML5" (Chrome, FireFox, Safari, IE9).
