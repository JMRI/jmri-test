La Table Mémoire

Vous pouvez utiliser le tableau de la mémoire pour regarder le contenu des variables de la mémoire JMRI.

Notez que vous pouvez trier le tableau par chacune des colonnes, cliquez simplement sur la tête de colonne.

Pour modifier la valeur stockée dans une mémoire particulière, il suffit de cliquer sur le valeur de colonne dans la ligne désirée. Entrez une nouvelle valeur, puis en appuyant sur le touche Retour ou Entrée pour stocker la valeur.

Pour créer une nouvelle variable mémoire, cliquez sur le bouton " Ajouter ..." au bas de la fenêtre.

Pour plus d'informations sur les mémoires, consultez le page d'aide mémoire.
