La Table Étiquette ID

     Vous pouvez utiliser la Table Étiquette ID pour regarder le contenu des objets IdTag JMRI.

Notez que vous pouvez trier la table par chacune des colonnes, cliquez simplement sur la tête de colonne.

Pour créer une nouvelle étiquette d'identification d, cliquez sur le bouton " Ajouter ..." au bas de la fenêtre.

Pour plus d'informations sur les étiquettes d'identification, consultez les page d'aide .
