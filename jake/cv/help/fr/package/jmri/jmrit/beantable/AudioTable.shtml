La Table Audio

Vous pouvez utiliser la Table Audio pour regarder le contenu des objets audio JMRI.

Notez que vous pouvez trier la table par chacune des colonnes, cliquez simplement sur le tête de colonne.

Pour créer un nouvel objet audio, cliquez sur le bouton " Ajouter ..." au bas de la fenêtre.

Pour plus d'informations sur les objets audio, consultez la page d'aide audio.
