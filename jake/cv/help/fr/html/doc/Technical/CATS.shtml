Integration avec CATS

Le Crandic Automated Traffic System (CATS) est une application basée sur JMRI crée par Rodney Black pour le contrôle et l'exploitation d'un chemin de fer miniature. Il y a plus d'informations sur le site web CATS .

JMRI est une bibliothèque, et CATS est une grande application maintenue séparément qui utilise la bibliothèque. En tant que tel, CATS constitue un important scénario de référence pour les développeurs JMRI car ils peuvent étendre et d'adapter leur code. CATS doit continuer à travailler pour ses utilisateurs, avec un minimum de 'extranous' effort pour les développeurs de CATS!

Integration pour test

CATS a deux ensemble d'arbres, "CATS .*»et «developper .*", correspondant à peu près aux deux applications CATS.

Le développement de CATS n'utilise pas CVS, et le code n'est généralement pas disponible pendant qu'il est en cours d'élaboration. Rodney Black communique régulièrement les disponibilités , y compris le code source, et les archives zip-file sur le site web CATS .

Depuis Juillet 2009, la distribution actuelle de CATS œuvres avec JMRI 2.6 et actuellement teste les version (2.7 .*) A partir de cette base, nous voulons garder les compilation CATS durant le développement de JMRI.
