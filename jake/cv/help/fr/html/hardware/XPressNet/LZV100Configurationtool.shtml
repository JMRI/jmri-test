'utilitaire de configuration JMRI LZV100

L'outil de configuration LZV100 est utilisé pour configurer la LZV100 travers une interface informatique LI100 ou LI101 .

Commençant par le haut, nous avons des informations de configuration pour la partie alimentation de la centrale de la LZV100.

Le menu déroulant sélectionne la tension piste.

La seconde liste déroulante de boites indique si le «E-Line" doit être active ou inactive. Quand elle est active l'alimentation des centrales utilise cette interface pour couper le réseau quand il y a un court-circuit, ou comme un arrêt d'urgence .

A partir du côté gauche de l'écran dans la première rangée de boutons, le bouton "Write to LZV100"(Ecrire à LZV100), écrira les réglages dessus de cette ligne dans la LZV100.

Le bouton "Reset to Factory Defaults"(Reset aux paramètres par défaut)" réinitialise l'écran d'usine par défaut. Ceci ne va pas envoyer une instruction de réinitialisation à la LZV100 .

Le dernier bouton, étiqueté "Fermer" va fermer l'écran, sans envoyer aucune information à la LI101F.

La deuxième rangée de boutons est des options pour la station de commande. ces options travaille aussi avec la LV100.

Le bouton "Reset Command Station"(Reset station de commande) envoie à l'adresse de locomotive 0 des instructions fonction indiquée dans le manuel LZV100 pour une commande de réinitialisation de centrale. Ceci permet de réinitialiser la centrale de commande aux parmàtres d'usine . Voir le manuel LZV100 pour plus d'informations pour savoir exactement ce que cela fait.

Le bouton «Auto/Manuel mode toggle" envoie à la locomotive à l'adresse 0 des instructions fonction indiqué dans le Manuel LZV100 pour basculer entre le mode de démarrage manuel et automatique. Dans le mode manuel , la LZV100 fonctionnera sur toutes les fonctions des locomotives qui étaient en service la dernière fois que la LZV100 a été allumée. En mode automatique, les locomotives reprendront également la vitesse et les réglages de direction précédents .
