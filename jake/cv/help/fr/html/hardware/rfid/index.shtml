JMRI: Support Lecteur RFID

Le support sur cette page est en rapide évolution; le code actuel peut être en avance ou en retard sur la documentation pour un jour donné.

JMRI fournit un support pour les Capteurs JMRI en général et Reporters en termes "d'évènements" du Lecteur RFID.

Désignation

La lettre du système lettre du système pour les connexions RFID est «F». Les détails de la RFID les noms d'événement et d'objets sont décrits ci-dessous.

JMRI associe les événements RFID avec des Objets individuels JMRI (Capteurs, Journals, etc) via le Noms système JMRI. Un nom système comme le "FSA" définit un capteur qui suit le lecteur "A" , tandis qu'un système comme le nom " FRB " définit un Journal qui suit le lecteur «B» .

Capteurs

Les lectures RFID dans les applications à venir de JMRI peuvent être accessibles via des objets Capteur JMRI. Le nom système du capteur détermine à quel lecteur RFID il correspond.

Un capteur est définie par deux événements: Celui qui l'ACTIVE, et celui qui le DÉSACTIVE. Quand une étiquette RFID se trouve dans la proximité d'un Lecteur RFID, le capteur associé est ACTIF; en dehors de ces moments, il est INACTIF. Vous créez les capteurs en utilisant le bouton Ajouter... sur la Table Capteur.

Journaux

Les lectures RFID dans les applications à venir JMRI peuvent aussi être accessibles via les Objets Journal JMRI. Le nom système du Journal détermine à quel lecteur RFID il correspond.

.Un journal peut contenir des informations arbitraires, contrairement à un capteur qui n'est qu'actif ou inactif. Chaque journal est associée à une source d'informations spécifiques sur le réseau, et fait que les informations périphériques soient disponibles pour le reste du JMRI.

Quand un lecteur RFID rapporte qu'une balise spécifique a été vue, le Journal correspondant contiendra des détails sur la vue spécifique IdTag . Ceci permet alors aux autres outils JMRI de réagir à cette information.

Table Journal

.La Table Journal fournit un accès facile au contenu de tous les Journaux que JMRI connaît. Il y a aussi un bouton «Ajouter ...» pour créer vos propres Journaux internes; JMRI crée automatiquement les Objets Journaux pour les appareils sur le réseau.

Affichage des Valeurs des Jounaux sur les Panneaux

La valeur d'un journale peut être affichée sur un panneau via une icône. Pour ce faire:
  • Sur la fenêtre Éditeur de panneau pour votre panneau, trouver le bouton "Ajouter Journal" .
  • Dans la boîte de texte à côté de ce bouton, tapez le nom utilisateur ou système du Journal désiré.
  • Cliquez sur le bouton. L'icône Journal sera placé sur le panneau. Si elle n'est pas visible, qui pourrait arriver si la valeur est actuellement vide; utiliser le tableau Journal pour changer la valeur à quelque chose qui sera visible.
  • Vous pouvez faire glisser l'icône à l'endroit où vous voulez de la manière habituelle.
  • Le menu pop-up sur l'icône vous permet de changer le formatage.
