Guide Matériel JMRI: Oak Tree Systems Railroad Control Interface (RCI)

Matériel Supporté

  • IO-24
  • IO-48
  • O-48

Limites

Les bibliothèques JMRI ont un support intégré pour les systèmes matériels série 'Oak Tree utilisant le protocole "RSC" bus. Les entrées peuvent être utilisées comme capteurs, et les sorties peuvent être conduites comme des aiguillages.

A cette époque (2008), le protocole de bus série RSI pour le régulateur HH-2 n'est pas encore pris en charge

Configuration
  1. Démarrez le programme JMRIdemo
  2. Dans le menu Edition, sélectionnez Préférences et configurer:
    • Protocole comme «Systèmes Oak Tree"
    • pour le port série, sélectionnez celui où votre matériel est connecté.
    • Sélectionnez la vitesse de transmission configurée dans vos cartes
    • Cliquez sur le bouton "Configurer les nœuds" . Sur le nouveau panneau qui s'ouvre:
      • Entrez une adresse de nœud (0 est généralement utilisé pour la première carte)
      • Sélectionnez le type de cartes à partir de la boîte d'initialisation
      • Cliquez sur "Ajouter Nœud"
      • Cliquez sur "Terminé"
    • Retour sur le panneau "Préférences ...", cliquez sur "Enregistrer", et dire oui quand il vous demande si vous voulez Quitter.
  3. Redémarrez le programme. Vos LEDs sur la carte devrait maintenant clignoter, indiquant que l'ordinateur et la carte dialogue.
  4. Pour vérifier que tout fonctionne, sélectionnez " Configuration Aiguillage "dans le menu Outils. Cela va ouvrir une petite fenêtre dans laquelle vous pouvez sélectionner un numéro d'aiguillage (voir voir ci-dessous ), puis cliquez direct/dévié pour changer l'état de cette sortie.
Utiliser avec plusieurs cartes

Si vous avez plusieurs cartes reliées au bus série du systèmes d'Oak Tree , simplement revenir à la fenêtre "configurer les nœuds" et de les saisir individuellement. Ne pas oublier d'enregistrer les préférences et redémarrer le programme.

Numérotation

Les noms système Oak Tree démarrent avec "O". Par exemple, "OS24" est un "S" Ensor (Capteur), la ligne d'entrée 24ème sur la carte du système Oak Tree avec adresse zéro .

Les entrées et sorties sont numérotées séquentiellement à partir de 1, sans se soucier des types de cartes, etc

Chaque carte est appelée un «nœud».les Capteurs et aiguillages sur le premier nœud sont numérotés de 1 à 999; les capteurs et les aiguillages sur le le second nœud sont numérotés de 1001 à 1999; etc

Par exemple:

  • Disons que vous avez une IO-48 avec son adresse interrupteurs réglé sur "2". Tous les entrées et sorties sur ce nœud seront numérotées à partir de 2001.
    • CT2001 est la première broche de sortie
    • CT2048 est la dernière broche de sortie
    • CS2001 est la première broche d'entrée
    • CS2012 est la dernière broche d'entrée

    Pour résumer, les entrées et sorties sont étiquetées de manière séquentielle au sein de chaque nœud. Les chiffres montent par 1000 de noeud à noeud. Les Numéros de nœuds peuvent aller aussi haut que le matériel le permet.
