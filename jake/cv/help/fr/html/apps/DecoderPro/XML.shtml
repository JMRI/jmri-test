JMRI: Information Décodeur

Le projet JMRI a rassemblé une grande quantité d'informations sur les Décodeurs DCC, il les met à la disposition du public de plusieurs façons. Elles sont décrites sur cette page.

Accès à l'information via le web

Les pages Web sont automatiquement créées à partir des fichiers de configuration DecoderPro dans le cadre de chaque version. Ceux-ci contiennent:
  • Identification des valeurs des CV du Décodeur utilisée par DecoderPro pour identifier le fabricant et le type d'un décodeur inconnu.
  • Index des pages individuelles décodeur , énumérées par le fabricant. Cela fournit un résumé des pages des caractéristiques CV et le contenu des décodeurs.
  • La sélection des guides annonce la taille décodeur , et par Caractéristiques électriques des décodeurs .
  • Résumé combiné des CV pour tous les décodeurs . Notez que c'est une très grande page!

Informations dans un format variable séparée par une virgule (CVS)

Pour rendre ces infos plus facile à importer dans un tableur ou un autre programme, elles sont également disponibles sous forme de fichiers Comma-Separated Variable (CSV). Fichiers avec données séparées par des virgules
  • la taille physique de chaque décodeur défini
  • CVs présent dans chaque décodeur
  • limites de puissance et connecteur par décodeur

Fichierss XML

La version définitive des données est dans des fichiers XML. Ceux-ci peuvent être obtenus auprès de toute distribution jmri, ou via le Web à l'adresse http://jmri.sf.net/xml.
