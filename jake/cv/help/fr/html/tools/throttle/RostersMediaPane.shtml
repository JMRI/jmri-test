Réglage des Listes Images et des Icônes



Le volet liste des médias est accessible lorsque vous utiliseze DecoderPro en mode avancé.



Entrée d'Image

La suppression d'un fichier image dans le cadre intérieur mettra à jour cette entrée (il doita être un fichier image, donc après avoir cherché sur le web pour une image que vous voulez, de l'enregistrer sur le bureau, puis déposer le fichier dans l'entrée, (déposer le contenu des images directement à partir de la fenêtre du navigateur ne fonctionne pas).

Le bouton Supprimer supprime l'entrée d'image en entrée de la liste actuelle.

Liste des images sont stockées dans <dossier JMRI Préférences >/ressources. Lorsque l'image est mise à jour ou retirée de la liste, le fichier image n'est pas retiré de son stockage dans <dossier JMRI Préférences >/ ressources.

l'entrée d'Image est utilisée comme un fond pour les limites JMRI lorsque cette option est activée.

Icône d'Entrée

Déposer un fichier image  dans le cadre intérieur mettra à jour cette entrée (il doit être un fichier image, donc après avoir cherché sur le web pour une image qui vous plaît, enregistrez-la sur le bureau, puis déposer le fichier dans l' entrée, laisser déposer le contenu des images directement à partir de fenêtre du navigateur ne fonctionnera pas ).
.
Le bouton Supprimer supprime l'entrée dans l'entrée liste Icône actuelle.

Liste des icônes estt stockées dans < dossier JMRI Préférences>/ressources. Lorsque l'icône est mise à jour ou supprimée de la liste, le fichier image n'est pas retiré de son stockage dans <dossier JMRI Préférences</ressources.

L'entrée icône est utilisé dans diverses zones de liste déroulante Liste de JMRI, et dans la Liste des fenêtres Régulateurs .

Il est recommandé d'utiliser des images GIF ou PNG avec un fond transparent et d'avoir les images tournées afin que la tête de locomotive soit à la droite. Cela permettra de faciliter les transformations de l'image dans des applications plus tard.

Remarque: un moyen facile de trouver des icônes est à la recherche pour "train GIF» ou regarder des ressources liée à la circulation " Screensaver "dans votre moteur de recherche Internet préféré".
