Barre d'outils Régulateur



La barre d'outils régulateur est activable depuis le Panneau des Préférences Régulateurs .

Lorsque elles sont activées, une Fenêtre Régulateur peut accueillir un ensemble de Régulateurs, ces Régulateurs peuvent être listés utilisant le:

bouton Précédent

Déplacer vers le Régulateur précédent pour cette Fenêtre de Régulateur.

bouton Suivant

Déplacer vers le Régulateur suivant pour cette Fenêtre de Régulateur.

Nouveau bouton

Ajouter un nouveau Régulateur dans cette Fenêtre de Régulateur.

Une fois que le nombre de Régulateurs dans une Fenêtre Régulateur est supérieure à un, ensuite les boutons Précédent et Corbeille sont activés.

Bouton Corbeille

Retirez le Régulateur actuel de cette Fenêtre de Régulateur.

Lorsque le nombre de Régulateur dans une Fenêtre de Régulateur est égal à un, les boutons Suivant, Précédent et Corbeille sont désactivés

Bouton d'arrêt

Envoyer un message d'arrêt à tous les Régulateurs gérés par JMRI.

Le bouton d'alimentation

Permet le contrôle de l'alimentation du réseau si possible (rouge est Arrêt, vert Marche , le jaune est le statut inconnu).

Bouton Liste

Afficher/Masquer la liste des fenêtre Régulateurs .
