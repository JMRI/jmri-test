JMRI: Panel Servlet

The JMRI Panel Servlets provide web access to information about "open" JMRI panels. This servlet responds to the following URL patterns:
  • /panel/Panel/<panel name>
  • /panel/ControlPanel/<panel name>
  • /panel/Layout/<panel name>
where Panel, ControlPanel and Layout refer to the JMRI editor used to create each type of panel.

The servlet returns an xml stream with various elements needed to recreate that particular panel externally.

Currently, there is a client-side web app called "showPanel" that uses these definitions to "draw" panels in a browser, allowing for clicks on active elements and update of changes made by this app and elsewhere in JMRI. [more info about showPanel]
