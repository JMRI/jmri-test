JMRI: TAMs Support

JMRI can connect to a TAMS Master Control command station using the built in serial interface.

The TAMs MC uses the Marklin p50x protocol.

This interface is still under development if you find any issues then please raise a issue on the yahoo group

To configure the connection:

  • Open the JMRI "Preferences" under the "Edit" menu,
  • Select "TAMS" as the manufacuturer, then connection type,
  • Select the appropriate serial interface.
  • The baud rate configured in the MC by default is 57600.
  • Click "Save".

Turnout, Programming, Throttle and power support is available for a JMRI-connected TAMs unit.

Consisting, Sensors and RailCom are not yet support.

Support for TAMS MC was first available in JMRI 2.99.7.
