L'outil JMRI XPressNet Système d'information

L'outil XPressNet Système d'information est utilisée pour déterminer quel matériel est connecté au système.

Les deux premières lignes indiquent le type de centrale de commande et les version des logiciels qui sont connecté à travers votre ordinateur. Cette information est lu dans le cadre du processus de configuration automatique lorsque JMRI débute la première fois, mais peut être relu à l'aide du bouton «Get System Info" (Obtenir Infos Système).

Après la lecture de l'interface de ligne, la version du matériel et des logiciels d' Interface ordinateur est affichée. Ces derniers sont obtenus par le "Get System Info" (Obtenir Infos Système).

L'image sur cette page montre les résultats obtenus en appuyant sur le bouton "Get System Info "avec une version du logiciel 2 LI101F connectée à un Atlas Commander avec le logiciel 3.2.

Le bouton "Fermer" va fermer l'écran.
