Linux installation instructions:

These steps outline how to install DecoderPro on Linux. These steps were tested on a system running Red Hat 7.2, but should work with all recent Linux distributions. These steps need to be performed with superuser priviledges.
  • Install Java Runtime Environment (JRE). To do this, download the Java 1.4 "Linux Red Hat RPM shell script" from http://java.sun.com/j2se/. (There is also a copy of the .bin for the file in this directory on the CD) This should give you a file named "j2re-1_4_0-fcs-linux-i386.rpm". Install that with: 
       rpm -i j2re-1_4_0-fcs-linux-i386.rpm
  • Copy the JMRI directory to your local disk.
  • Install portions of Java Communications API. Download the file "javacomm20-x86" from http://java.sun.com/products/javacomm/index.html (Licensing restrictions prevent us from putting a copy of this on the CD) Then install it with 
              
   cp comm.jar /usr/java/j2re1.4.0/lib/ext/
   echo Driver=gnu.io.RXTXCommDriver > /usr/java/j2re1.4.0/lib/javax.comm.properties
  • Install Rxtx. Download the file "rxtx-bins.1" from http://www.rxtx.org. (There is also a copy on the CD). Install it with 
        cp rxtx/1.4/jcl.jar /usr/java/j2re1.4.0/lib/ext
    cp -d rxtx/1.4/i386-pc-linux/lib* /usr/java/j2re1.4.0/lib/i386/
  • Give user-level application access to serial ports 
       chmod 666 /dev/ttyS0
   chmod 666 /dev/ttyS1
  • Give user access to lock files (required on Red Hat). Edit /etc/groups & /etc/gshadow to add user to 'uucp' and 'lock' groups.
The install is complete. Switch to the JMRI directory and run the programs using one of 
  ./DecoderPro.csh
  ./LocoTools.csh
  ./JmriDemo.csh

Connecting your layout to the computer

To connect your layout to the computer, follow one of these links:
  • Atlas Commander via Lenz LI100
  • EasyDCC
  • Digitrax
  • NCE
  • Lenz XPressNet via LI100
