Macintosh MacOS X information:

MacOS X version 10.1.3 or later is required.

To install

You will need drivers for the serial hardware you're using. We test using a Keyspan PDA Adapter; those drivers are available at http://www.keyspan.com/support/macosx. Some device drivers will list each port under several names, e.g. starting with "/dev/tty" or "/dev/cu", for example "/dev/tty.KeyUSA19181.1". In that case, you must select the one that starts with "cu", e.g. "/dev/cu.KeyUSA19181.1".

You also need the javax.comm package. Download the installer for this from http://prdownloads.sourceforge.net/jmri/JavaCommInstaller2.hqx?download. Double click on the downloaded and expanded file to install it. You will need the MacOS X administrator password to run the installer.

Note that you will have to reinstall the driver and javax.com package if you have to reinstall the system software, or if you do a "clean install" of an OS update.

Download the 1.2.5 version from http://prdownloads.sourceforge.net/jmri/JMRI.1.2.5.hqx?download and unpack it. This will create a "JMRI" folder.

To install, you just have to move the JMRI folder to where you want it on your computer. You can then run the program by double-clicking on the "DecoderPro", "LocoTools", "PanelPro" or "JmriDemo" application icons in the JMRI folder.

The 1.2.5 version is our most recent "production-quality" version. The project is continuously adding features, however, and new "test" versions appear every couple of weeks. You may find one of these has features that you really want. These are announced in the "jmriusers" Yahoo discussion group at http://groups.yahoo.com/group/jmriusers/. To download one of these, please see the test version download page.

Connecting your layout to the computer

To connect your layout to the computer, follow one of these links:
  • Atlas Commander via Lenz LI100: Note that you can't program decoders with this configuration.
  • Digitrax
  • EasyDCC
  • Lenz XPressNet via LI100
  • NCE
  • SPROG
  • Zimo MX-1

If your Macintosh has the original "mini-DIN-8" serial connectors, please see our page on wiring an adapter cable

You can also go directly to the pages describing how to use the DecoderPro program, use the LocoTools program, and the Jmri libraries and JmriDemo program.
