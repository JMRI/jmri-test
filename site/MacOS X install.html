Macintosh MacOS X information:

MacOS X version 10.1.3 or later is required.

To install

You will need drivers for the serial hardware you're using. We test using a Keyspan PDA Adapter; those drivers are available at http://www.keyspan.com/support/macosx. Some device drivers will list each port under several names, e.g. starting with "/dev/tty" or "/dev/cu", for example "/dev/tty.KeyUSA19181.1". In that case, you must select the one that starts with "cu", e.g. "/dev/cu.KeyUSA19181.1".

You also need the javax.comm package. Double click on the "Java comm installer" file to install it. You will need the MacOS X administrator password to run the installer.

Once those packages are installed, you can run the program from the CD by double-clicking on the "DecoderPro", "LocoTools" or "JmriDemo" application icons in the JMRI folder.

You can also copy the entire JMRI folder to your local disk and run it from there.

Connecting your layout to the computer

To connect your layout to the computer, follow one of these links:
  • Atlas Commander via Lenz LI100
  • EasyDCC
  • Digitrax (Note that MacOS X cannot currently communicate with a Digitrax MS100)
  • NCE
  • Lenz XPressNet via LI100

You can also go directly to the pages describing how to use the DecoderPro program, use the LocoTools program, and the Jmri libraries and JmriDemo program.
