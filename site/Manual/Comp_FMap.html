Some decoders allow you to change the function assigned to each set of output wires. When this capability is present, the checkboxes in the window below are active and allow you to choose what function key on the throttle each output responds to.

Multiple choices for function assignments would be shown by several checkboxes. This is more common in decoders with more than four functions, and in sound decoders. A single dimmed checkbox like those above indicates that no changes in function assignments are possible.
