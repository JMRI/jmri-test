Some decoders allow you to change the function assigned to each set of output wires. Multiple choices for function assignments are shown by several checkboxes. This is more common in decoders with more than four functions, and in sound decoders. When this capability is present, the checkboxes as shown in the window below are active and allow you to choose to which function key on the throttle each output responds.

A single dimmed checkbox like those shown in the excerpt below indicates that no changes in function assignments are possible.
