La Table L-Itinéraire

Un L-Itinéraire est une version plus généralisée d'un itinéraire mis en œuvre en tant que Logix. Il est destiné à être utilisé dans des situations où des déclenchements supplémentaires ou actions sont souhaités au-delà des capacités des Itinéraires. Il peut être utilisé comme point de départ pour créer de nouveaux Logix. Le tableau L-Itinéraire fournit les listes de sélection des Capteurs, des Aiguillages, Lumières et Têtes de signal à partir de desquels les actions de sortie et les conditions de déclenchement peuvent être choisies.

Une fois que vous avez quelques aiguillages, capteurs, têtes de signal et/ou les lumières de gérer, utiliser le bouton Ajouter dans le coin inférieur gauche de la table L-Itinéraire pour créer un nouvel itinéraire Logix. Une fois que vous avez créé un itinéraire Logix, vous pouvez ajouter d'autres capacités en l'éditant depuis la Table Logix.

Logix qui implémente le L-Itinéraire a son nom système préfixé par «RTX». Tous les Logix avec ce préfixe sont affichés dans le tableau L-Itinéraire. S'il vous plaît noter qu'une «RTX» Logix qui a eu ces capacités ajoutées avec Logix édition ne peut pas être capable d'afficher ces capacités additionnelles pour l'édition de L-Itinéraire. Une fois l'édition Logix est faite pour un L-Itinéraire il est recommandé que les autre éditions soient faites à partir du tableau Logix.

Colonnes Tableau L-Itinéraire

Nom du système
Le Nom Système ne peut pas être changé une fois entré dans la table. Le bouton Ajouter situé dans le coin inférieur gauche de la table vous permet d'ajouter un nouvel L- Itinéraire à la table.
Nom Utilisateur
Vous pouvez directement saisir ou modifier le champ "Nom Utilisateur" pour un titinéraire Logix existan
Activé
Pour désactiver un L-Itinéraire à partir de déclenchement, décochez la case "Activé" pour l'itinéraire . Ceci définit son Logix à un état désactivé. Le L-Itinéraire peut être activé ou désactivé soit à partir de la table ou du tableau L-Itinéraire Logix.
 
Modifier
Appuyez sur ce bouton pour modifier la L-Itinéraire . Pour plus d'informations, voir: Ajout/Edition Litinéraires.

Contrôles de la Table L-Itinéraire

Il y a un bouton au bas de la Table L-Itinéraire
Ajouter
Appuyez sur ce bouton pour ajouter un nouvel itinéraire Logix. Pour plus d'informations, voir Ajouter/Editer L-Itinéraires.
