Cette page décrit les détails techniques du soutien de JMRI pour CBUS Voir aussi les pages sur les JMRI soutien général CAN .

CAN adaptateurs

L'approche générale JMRI est de soutenir le plus grand nombre de combinaisons de matériel possible. En soutenant le plus grand nombre possible de personnes et leurs matériels, nous pouvons créer une communauté grandissante de développeurs, donc plus de code.

Dans le cadre de la présente, le soutien JMRI CAN est conçu pour utiliser autant de types d'adaptateur CAN que possible..

Le MERG CAN-RS peut faire "l'énumération " pour CBUS, mais de nombreux autres adaptateurs ne peuvent pas être utilisés en raison de leur latence intrinsèque. Leurs réponses ralentit efficacement et les rend invisibles.

Par conséquent, Mike Bolton, l'un des développeurs du CBUS a suggéré que JMRI utilise un ID CBUS de 126. Actuellement, c'est une valeur fixe.
