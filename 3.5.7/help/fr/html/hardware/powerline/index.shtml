JMRI: Contrôleurs CPL (Powerline)

Cette page explique comment configurer JMRI pour parler aux contrôleurs de la ligne électrique, par exemple X10.

Voir aussi la page de compatibilité associée, pour voir les interfaces prises en charge.

Pour plus d'informations sur la façon de répondre à vos équipements via JMRI, s'il vous plaît voir la page sur le traitement et les noms de JMRI pour X10.

Les lumières JMRI travaillent de manière intuitive avec des appareils CPL. Lorsque la Lumière JMRI est mis sur "ON", une commande est envoyée au périphérique distant pour le mettre sur "ON". Idem pour "OFF".

Il y a aussi un support limité pour la gradation (luminosité variable) Lumières. Voir la page "Contrôle-Lumière" pour plus d'informations à ce sujet.

L'aiguillage JMRI est disponible pour X10. Lorsque l'aiguillage JMRI est réglé sur "dévié", une commande est envoyée à l'appareil à distance qui est réglé à "BRIGHT". Ceci est une pleine puissance ,immédiate. Lorsque l'aiguillage est réglé sur "direct", au périphérique distant est envoyé une commande pour aller sur "OFF".Selon le matériel X10, ce peut être soit un arrêt immédiat, ou être baisse lente.
