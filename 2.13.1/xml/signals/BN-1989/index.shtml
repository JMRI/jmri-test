JMRI: BN-1989 Signaling Definition

This directory contains signaling definitions corresponding to the BN (Burlington Northern) rules as described in the October 1989 Burlington Northern Railroad Southern Region, Debraska Division Timetable #2. In effect at 0001 Continental Mountain Time, Sunday October 29, 1989.

Per Burlington Norhtern General Signal Instructions, Dwarf signals display the same aspects and indications as high signals.

See the aspect page.
