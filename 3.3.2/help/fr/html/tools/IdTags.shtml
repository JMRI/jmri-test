JMRI: Documentation Balise d'Identification

Id Tag

Une balise d'identification est le moyen pour JMRI de représenter des informations sur les étiquettes d'identification qui peuvent être attachés à, par exemple, divers éléments de matériel roulant.

Quand un balise d'identification est lue par un lecteur approprié, cela est notifié par le biais à la fois d'un Capteur qui devient actif et un Journal où est fixé la valeur de la balise d'identification. La balise ID sera également enregistrer à la fois par le Journal qui l'a vue la dernière fois, ainsi que l'heure à laquelle elle a été vue la dernière fois

Si une étiquette d'identification inconnue est détectée par un lecteur, les détails de cette étiquette seront automatiquement inscrits dans la table étiquette d'identification .

À l'arrêt du système, les détails de tous les ID Tags connus du système sont écrits dans un fichier stocké dans le sous-répertoire "idtags" des préférences des utilisateurs. Le stockage des détails quand et où une étiquette d'identification donnée a été vu la dernière fois est configurable en mettant l'option au bas de la table étiquette d'identification.

La mise en œuvre initiale est pour MERG Les lecteurs RFID, à la fois autonome et via un concentrateur.

Table Étiquette d'Identification

la table étiquette d'identification fournit un accès facile au contenu de tous les étiquettes ID que JMRI connaît. Il y a aussi un bouton «Ajouter ...» pour créer votre propre étiquette ID, JMRI crée automatiquement les Objets d'étiquette d'identification pour les balises détectées sur le réseau.

Afficher les étiquettes d'identification sur les panneaux

Une étiquette d'identification peut être affichée sur un panneau via l'utilisation de l'icône Journal associé à un lecteur donné. Pour ce faire:
  • Sur la fenêtre Éditeur de panneau pour votre panneau, trouver le bouton " Ajouter Rapport".
  • Dans la boîte de texte à côté de ce bouton, tapez le nom utilisateur ou système du Rapport désiré.
  • Cliquez sur le bouton. L'icône Rapport sera placé sur le panneau. Si elle n'est pas visible, ce pourrait être parce que la valeur est actuellement vide; utiliser le tableau Reporter pour changer la valeur à quelque chose qui sera visible.
  • Vous pouvez faire glisser l'icône à l'endroit où vous voulez avec la manière habituelle.
  • Le menu pop-up sur l'icône vous permet de changer le formatage.

Plus d'informations sur les Journaux peuvent être trouvés dans la page d'aide Journaux.
