Ajout/Modification des Objets Audio

Pour créer un objet audio, cliquez sur le bouton "Ajouter ..." sur l Table audio. Cela ouvrira une petite fenêtre avec un espace pour entrer un nom système et nom 'tilisateur pour la nouvelle mémoire, plus et bouton "Ajouter".

Vous devez entrer un nom système unique. Le système des noms pour les objets audio ont un ensemble de motif. Ils commencent par "IAL" pour "Listener audio interne", "IAB" pour "Audio Buffer interne", ou "IAS" pour "Source audio interne".

Il ya des fenêtres séparées pour l'ajout de chacun des différents types.

Le nom utilisateur est facultatif, mais si on leur en donne un, il doit être unique.

Pour plus d'informations sur les objets audio, consultez la page d'aide audio.
