Volet des préférences Régulateurs



Pour une meilleure expérience, le choix recommandé est cellui illustrée ici.

Ce panneau de préférences est accessible à partir des sous-menus Régulateurs dans les sous-menus Régulateur dans l'entrée du menu Outils .

Il est recommandé de fermer toutes les fenêtres Régulateurs avant d'appliquer une nouvel ensemble de préférences, puis rouvrez les une fois les nouveaux paramètres appliquée ou sauvés.
d'entrée
  • Le bouton Annuler ferme la fenêtre et laisse les paramètres actuels des régulateurs comme ils sont.
  • d'entrée Le bouton Appliquer appliquera l'ensemble actuel des préférences et la fenêtre des préférences d'entrée restera ouverte.
  • d'entrée Le bouton Enregistrer enregistrera vos préférences actuelles, et fermerera les fenêtres Préférences, les réglages seront chargés automatiquement au prochain démarrage du JMRI.

    • Utilisez Régulateur étendu

    Ce réglage active/désactive toutes les fonctionnalités étendues des régulateurs.

    Réglez Contrôles transparents

    Mettra le fond de la fenêtre interne du régulateur transparente ou semi- transparente.

    Cette fonctionnalité est appelée à bien travailler sur OS X, Lin d'entrée ux et Windows, mais pour être travaillé, varie totalement selon l'interface actuelle Java (modifiable dans préférences principales JMRI )

    Utiliser l'image du fichier comme fond

    Quand une liste est sélectionnée, l'image définie dans son Panneau des médias sera chargée comme un fond pour la fenêtre de commande du régulateur.

    Lorsque cette option est activée et l'adresse sélectionnée est 0 ou 3, une image par défaut sera chargée.

    Redimensionnez la fenêtre à la taille de l'image

    Quand une image de fond est chargée pour les régulateurd'entrée s actuels, va forcer la taille de Fenêtre du Régulateur à celle de cette image.

    Recherche liste infos quand adresse indiquée

    Quand une adresse est inscrite dans l'AdressPanel , la base de données JMRI recherchera une entrée de registre correspondante (et sera donc des options de déclenchements décrits ci-dessus et ci-dessous s'ils sont activés).

    Chargement automatique de la fenêtre de préférences régulateurs lorsqu'un lien est trouvé avec la Liste

    Quand une liste est sélectionnée, si un fichier de définition régulateur correspondant existe, JMRI le chargera automatiquement.

    Ces fichiers sont stockés dans <JMRI Preference Folder>/Throttle/XYZ.xml où XYZ dans l'ID de la Liste.
    Utiliser le entrée menu Enregistrer du Menu Fichier dans une fenêtre régulateur va créer ou mettre à jour ces fichiers.

    Ignorer position du cadre Régulateur

    Pour les vieux fichiers régulateurs qui contiennent des position de Fenêtre Régulateur en eux. Faites comme décrit.

    Remarque: les positions des Fenêtre Régulateurs sont maintenant sauvegardées dans le mise en page RégulateurFichiers de sauvegarde (pas dans n'importe quels fichiers de sauvegardes Régulateurs )

    Masquage indéfini de la Liste Boutons

    Quand une liste est sélectionnée, les boutons des panneaux d'adresse sont initialisés comme définis dans le panneau de Fonctions dans DecoderPro, si cette option est activée, les boutons ne sont pas définis (sans texte) au PanelPro seront automatiquement cachés dans le Panneau Fonction Panneau Fonction .

    Remarque: vous pouvez réinitialiser la visibilité des Fonctions Boutons dans une fenêtre régulateur en allant dans Menu Afficher et cliquer sur "afficher tous les boutons de fonction" .

    Activer la barre d'outils papillon

    d'entrée Cette option activera une barre d'outils qui apparaîtra dans toutes les Fenêtres Régulateurs .
