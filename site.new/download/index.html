Latest News

Production version 1.8 released

On July 22, 2007, production version 1.8 was made available.

Version 1.8 replaces version 1.6.1 as the current "production" version, recommended for first-time users. Please see the next section for download and install information.
Test version 1.9.1 released

Test version 1.9.1 was made available on November 4, 2007.

JMRI 1.9.1 is the most recent test release, containing the latest and greatest JMRI updates. Some key changes include
  • a new layout editor for creating panels,
  • NCE PowerCab support,
  • lots of new and updated decoder definitions, and
  • the way that layout information and panels is stored has been improved
along with lots of other changes. For more information, including the links to download a copy, please see the release note. Version 1.8 is the current "production" version, recommended for first-time users.

JMRI 1.8 - Production release

If you've never installed a version of this program before, please read the installation guide (on the left) for your computer type and layout hardware. They will show you how to install the things that are needed before the program will run.

Installation guides: Windows, MacOS X, Linux

Layout connection: Atlas, C/MRI, Digitrax, EasyDCC, Lenz, NCE, Roco, SPROG, Uhlenbrock, Wangrow, Zimo

If you've previously installed a working version of the program, you only need to download and install one of the files listed here.

Download:

  • MacOS X: http://downloads.sourceforge.net/jmri/JMRI.1.8.dmg?download
  • Windows: http://downloads.sourceforge.net/jmri/JMRI.1.8.exe?download
  • Linux: http://downloads.sourceforge.net/jmri/JMRI.1.8.tgz?download

Notes:

If you're looking to download Rodney Black's CATS application, see the CATS download page.

JMRI - Developer releases

As the program develops, we also produce test versions. You may want to try these, as new features will show up in them first. They are announced in the JMRI users email group.

JMRI Developer Release 1.9.1

Test version 1.9.1 was made available on November 4, 2007.

Some key changes include a new layout editor for creating panels, NCE PowerCab support, lots of new and updated decoder definitions, and the way that layout information and panels is stored has been improved.

For the complete list of changes and download links, see: JMRI 1.9.1 Release Notes

JMRI Developer Release 1.7.7

Test version 1.7.7 was made available on June 9, 2007.

Some key changes include numerous decoder updates, improvements to support for the new NCE EPROM, including a send macro tool and turnout readback, improvements to signaling logic, Routes and the new Logix capability for layout control, a complete revamp of help system (including help information for most windows) greatly improved support for the Digitrax PR2 programmer and it's sound files,

For the complete list of changes and download links, see: JMRI 1.7.7 Release Notes

JMRI Developer Release 1.7.6

Test version 1.7.6 was made available on January 24, 2007.

Some key changes include numerous decoder updates, improvements to signaling logic, and greatly improved support for the Digitrax PR2 programmer and it's sound files.

For the complete list of changes and download links, see: JMRI 1.7.6 Release Notes

JMRI Developer Release 1.7.5

After an unfortunate delay of several months due to events beyond our control, test version 1.7.4 was made available on July 1, 2006. This was quickly followed on July 4 with version 1.7.5 to fix some bugs uncovered in the first testing.

Some key changes include additional support for sound decoders, support for Oak Tree Systems boards and the Digitrax PR2 programmer, and other great stuff.

For the complete list of changes and download links, see: JMRI 1.7.5 Release Notes
