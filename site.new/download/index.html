Latest News

Test version 2.5.7 released

Test version 2.5.7 was made available on June 27, 2009.

JMRI 2.5.7 is the most recent test release, containing the latest and greatest JMRI updates.

This is the seventh of a series working toward an eventual JMRI 2.6 production version.

For more information, including the links to download a copy, please see the release note. Version 2.4 is still the current "production" version, recommended for first-time users.

Production version 2.4 released

On December 21, 2008, production version 2.4 was made available.

Version 2.4 replaces version 2.2 as the current "production" version, recommended for first-time users. Please see the next section for download and install information.

JMRI 2.4 - Production release

If you've never installed a version of this program before, please read the installation guide (on the left) for your computer type and layout hardware. They will show you how to install the things that are needed before the program will run.

Installation guides: Windows, MacOS X, Linux

See also the page of layout connections.

If you've previously installed a working version of the program, you only need to download and install one of the files listed here.

Download:

  • MacOS X: http://downloads.sourceforge.net/jmri/JMRI.2.4A.dmg?download
  • Windows: http://downloads.sourceforge.net/jmri/JMRI.2.4A.exe?download
  • Linux: http://downloads.sourceforge.net/jmri/JMRI.2.4A.tgz?download

Notes:

If you're looking to download Rodney Black's CATS application, see the CATS download page.

JMRI - Developer releases

As the program develops, we also produce test versions. You may want to try these, as new features will show up in them first. They are announced in the JMRI users email group.

For more information on the rough development schedule, please see the JMRI road map.
JMRI Developer Release 2.5.7

Test version 2.5.7 was made available on June 27, 2009.

Some key changes include

  • There's now a new and improved Panel Editor, along with the ability to define and save your own sets of icons.
  • Another round of improvements to the operations feature
  • Improved handling of errors when loading a panel, which should make it much less likely that people will overwrite their panel files with ones that are missing data.
  • Panel Editor users may now change the default icons to be icons of their own choosing.
  • Fix to a problem editing Signal Head definitions from their table.

For the complete list of changes and download links, see: JMRI 2.5.7 Release Notes

JMRI Developer Release 2.5.1

Test version 2.5.1 was made available on January 15, 2009.

Some key changes include

  • New DecoderPro speed table controls that allow you to quickly setup your loco.
  • Updates to the CTI Acela connection.
  • This release has many improvements to the operations program. There's now a demo program to introduce you to some of the capabilities of the operations program. You can now quickly make a copy of a route. New controls allow you to decide whether cars are picked up and/or dropped off at the various locations in a route. A new feature allows you to schedule the delivery order of cars to a siding. And you can now optionally add the operations menu to the main menu bar.

For the complete list of changes and download links, see: JMRI 2.5.1 Release Notes

CATS

The Crandic Automated Traffic System (CATS) is a JMRI-based application by Rodney Black for controlling and operating a model railroad. There's more information on the CATS web site.

Version 40 of CATS (current December 1, 2007) requires JMRI 1.8; the test releases in the JMRI 1.9 series are not recommended at this time for use with CATS.
