JMRI: Technology Road Map

This page is a draft road map for JMRI's future use of Java technologies. Nothing here is definite (as of July 2007), and discussion is continuing on the jmri-developers mailing list.

JMRI Releases

  • 1.8 - production version, culmination of 1.7.* series
  • 1.9 - development series. Move forward to Java 1.X, remove Java 1.1.8 compatibility code.
  • 2.0 - production version, culmination of 1.9.* series (target date: ....)
  • 2.1 - development series.
