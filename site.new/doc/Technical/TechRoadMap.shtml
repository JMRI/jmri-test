JMRI: Technology Road Map

The 2007 user poll seems to indicate that there are very few people using Windows 95 or Macintosh "Classic" any more. This is important because there only Java 1.1.8 is available for the Macintosh operating system, and only Java 1.3 is available for Windows 95.

See the page on Java Capabilities for more info on what features are available starting in which versions of the Java Runtime (JRE) and Java Development (JDK) environments.
