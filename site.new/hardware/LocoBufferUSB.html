JMRI Hardware Guide: Connecting a LocoBuffer-USB to LocoNet

The LocoBuffer-USB provides a very reliable way to connect a computer to a LocoNet. It is produced by Dick Bronson's RR-CirKits company, based in part on the original LocoBuffer design.

Recommended setup

The recommended setup for the LocoBuffer-USB is:

  • Set all switches off (toward the front label side).
  • Start DecoderPro or other program and configure it for:
    • LocoNet LocoBuffer-USB
    • Select your port

Loop-back testing

The LocoBuffer-USB input circuit and code actually reads from the LocoNet output jack, so if you install the LocoNet terminator (the small circuit that was shipped with the LocoBuffer II) and set SW7-8 to ON (down) so that the input circuitry gets it's power from the wall-wart, then you have created a loop back test. The only thing that is not tested is the LocoNet jack itself.

Steps:

  1. Unplug the LocoNet cable
  2. Open case (use a small screw driver to work open the latches at the small slots on the sides)
  3. Install terminator
  4. Set Sw 7-8 ON, all other switches OFF
  5. Open the JMRI LocoNet Monitor window.
  6. Using JMRI send some commands (e.g. switch commands)
  7. The commands should appear in the LocoNet monitor window and the LocoBuffer-USB activity LED should blink
  8. Remove the terminator (unless your application requires it)
  9. Restore the LocoBuffer-USB option switches to their normal settings for your application

LocoBuffer-USB Power

When powered by an external supply, the LocoBuffer-USB completely isolates your computer from the LocoNet, removing the possibility of ground currents flowing through the LocoNet cable.

If you are using an isolated computer, for example a laptop that's not connected to any peripherals, you can configure the LocoBuffer-USB so that it doesn't need an external power supply. It will then get it's power from the RailSynch leads in the LocoNet.

It is strongly advised that you get your computer connection working properly with an external power source before you do this. That way, if any problems arise when you try to use internal power, you'll know what caused them and they'll be much easier to diagnose.

To do this, remove the power connection and set switches 7 and 8 to ON (down, toward the switch illustration).
