JMRI Partners

JMRI works with other software in various ways.

For example, Giorgio Terdina has written the XTrkCadReader tool to let you use your XTrkCad layout designs to create JMRI layout panels.

Rodney Black's CATS is a modern computer-based dispatching system that works with JMRI to connect to the layout.

Tom Stack's Model Railroad Manager is integrated system for model railroad inventory, operations, control, dispatching and automation. It can access local or remote JMRI-connected layouts.
