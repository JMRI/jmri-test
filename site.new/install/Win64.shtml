JMRI Install Guide: 64-bit Windows

This page is a start at documenting how to use JMRI on 64-bit Windows. It is still under construction.

JMRI runs in 32 bit mode, even on 64-bit Windows systems.

Even after installing Java on a Windows Vista 64-bit computer, JMRI applications won't start because the "java" command can't be found.

Two Sun links that might help:

  • http://java.sun.com/javase/6/webnotes/install/jre/install-windows-64.html
