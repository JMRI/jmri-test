JMRI: Decoder Information

The JMRI project has collected a large amount of information about DCC decoders, which it makes publicly available in several ways. These are described on this page.

Access to the information via the web

Web pages are automatically created from the DecoderPro configuration files as part of each release. These contain:
  • Decoder identification CV values used by DecoderPro to identify the manufacturer and type of an unknown decoder.
  • Index to individual decoder pages, listed by manufacturer. This provides a one-page summary of the characteristics and CV contents of the decoders.
  • Selection guides listing decoder size, and by decoder electrical characteristics.
  • Combined summary of CVs for all decoders. Note that this is a very large page!

Information in Comma-Separated-Variable (CSV) format

To make it easier to import into a spread-sheet or other program, these are also made available as files in Comma-Separated-Variable (CSV) format.
  • decodersizes.csv Physical size for each defined decoder
  • cvsummary.csv CVs present in each decoder
  • decoderpower.csv Power limits and connector by decoder

XML files

The definitive version of the data is in XML files. These can be obtained from any jmri distribution, or via the web at http://jmri.sf.net/xml.
