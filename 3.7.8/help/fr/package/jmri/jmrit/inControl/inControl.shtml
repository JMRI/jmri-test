JMRI: inControl - Régulateurs Web et Fenêtr es (inControl.html application)

Cette application est une page web unique qui fonctionne sur les navigateurs les plus récents dans les ordinateurs, smartphones et tablettes.
Il n'y a rien à installer!

Pour commencer à utiliser JMRI via un navigateur web, il vous suffit d'activer le Serveur Web JMRIinclus. Vous pouvez trouver des informations supplémentairesici.

Il commande non seulement les locomotives et décodeurs de fonction, comme un régulateur web, mais aussi les aiguillages, les lumières et autres appareils à travers les fenêtres JMRI actives.

inControlest entièrement redimensionnable et ajustes automatiquement la taille et l'orientation de l'écran de l'appareil.

N'importe qui peut créer des pages web avec les iframes contenant inControl.html, afin de créer des panneaux de commande plus ou moins complexes.

Paramètres de chaîne de requête

Presque tous les paramètres peuvent être transmis à l'application à travers la chaîne de requête, mais ils peuvent être modifiés dans les sections Réglages et Paramètres Loco/Appareil.

Réglages

Cette section permet de modifier les paramètres globaux. Ils sont constamment stockés dans le stockage local du navigateur.

Paramètres Loco/Appareil

Cette section permet de modifier les paramètres du décodeur de la locomotive ou de la fonction. Ils sont constamment stockés dans le stockage local du navigateur.

Commande Vitesse peut être désactivée. Ceci est approprié pour contrôler une fonction uniquement décodeur.

Pour activer une touche de fonction, Active doit être activé.

Si une touche de fonction ne s'active pour certaines action que lors d'une transition de OFF à ON, Auto unpress doit être activé.

Pour contrôler un décodeur standard et des fonctionseulement décodeurs simultanément avec le même régulateur, Adresse Alternative Appareil peut être réglé à l'adresse de la fonction seulement décodeurs.

Pour donner une touche de fonction pour activer manœuvre, Shuntdoit être activé. Il modifie le comportement de la commande de vitesse afin de permettre de contrôler de -50 % à +50 % de la vitesse maximale. Quand positif, la locomotive se dirige vers la direction choisie. Le Bouton bascule devrait être activé.

Balayer pour être en commande ( application Scan2BeInControl.html)

Il suffit de diriger le smartphone et commencer à commander une locomotive ou une trame!

Cette application énumère la liste (locos) et les images définies dans JMRI .

Pour ouvrir un régulateur web ou une image à l'aide de inControl, les utilisateurs peuvent:
- numériser l'image en utilisant un smartphone (avec application de lecteur compatible qrCode)
- Cliquez ou touchez la Loco ou l'image ou le lien

Pour les locos, les paramètres de JMRI sont fusionnée avec ceux déjà stockés dans votre appareil (ordinateur, tablette ou smartphone) .
Si vous souhaitez utiliser uniquement les paramètres de JMRI, vous devez supprimer la loco à partir de votre appareil avant de numériser le qrCode graphique ou utilisez le lien .
