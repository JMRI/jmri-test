Welcome to JMRI TrainPro.

Note to developers: this really should be a quick start with the default tool bar items explained in a manner similar to the jMonkeyEngine SDK welcome page.

TrainPro is an application for managing and running a DCC-equipped model railroad.

Migrating from DecoderPro or PanelPro

TrainPro uses the same preferences files as DecoderPro and PanelPro, so TrainPro will, in most cases, "just work" with the other applications, and you should be able to switch freely back and forth between them. There are a few exceptions, however:

  • Startup buttons are now listed in a tool bar. The tool bar can be customized by right-clicking on it and selecting Customize. Changes made this way are ignored by DecoderPro and PanelPro; they are also not specific to a single configuration profile.
