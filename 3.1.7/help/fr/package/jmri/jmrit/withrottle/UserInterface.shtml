JMRI: Régulateur WiFi

Cette fenêtre contrôle la connexion entre les manettes WiFi, par exemple un IPhone, IPod Touch, ou téléphone Android, et JMRI.

Il peut être configuré pour démarrer automatiquement avec le " Démarrage " dans la section " JMRI Préférences " section Préférences JMRI sous la rubrique «Actions».

Les Préférences pour les manettes WiFi peuvent être trouvées dans les Préférences JMRI sous la rubrique RégulateurWi.

Cet utilitaire diffuse ses informations de connexion à travers le réseau local afin qu'il puisse être capté par un dispositif à la recherche du service. Ceci élimine d'avoir à entrer vos informations de connexion dans l'appareil. Si l'ordinateur exécutant JMRI est protégé par un logiciel pare-feu , vous pouvez avoir besoin de désactiver le pare-feu pour permettre aux Régulateurs WiFi de se connecter.

Lorsque vous utilisez un régulateur WiFi, si l'ordinateur passe en mode veille, le contrôle sera perdu. Il peut être nécessaire d'ajuster les préférences du système pour empêcher l'ordinateur de se mettre en veille quand il est inactif.

Pour utiliser votre IPhone ou IPod Touch avec JMRI comme un régulateur, vous exécuter les applications sur la page WiThrottle . de l'IPhone ou l'IPod Touch. L'application de de base est disponible en téléchargement gratuit surl'ITunes Store.

Pour utiliser votre téléphone Android avec JMRI comme un régulateur, vous exécuter le programme Pilote Machine sur le téléphone Android.

Le protocole est décrit sur une page separée. Pour en savoir plus sur JMRI en réseau voir la page technique sur le réseau local.

Étiquettes Fonctions

Étiquettes Fonctions peuvent être personnalisées pour chaque fiche à travers JMRI . Ces Étiquettes peuvent être affichées à la place des des génériques F1, F2...sur l'écran de l'équipement mobile. Les Étiquettes peuvent être ajoutées quand une nouvelle fiche est créées, ou par l'édition d'une fiche existante sous l'onglet Étiquettes Fonctions. Pour entrer des étiquettes de fonction, le format du programmateur doit être soit "Avancé" soit "Personnalisé".

Itinéraires et Aiguillages

L'application peut contrôler tous les aiguillages (et itinéraires) que JMRI contrôle. S'is sont disponibless dans la table aiguillage (ou itinéraire) de JMRI, ils sont automatiquement disponibles des les régulateurs WiFi.  S'il n'y a pas d'aiguillages (ou itinéraires dans la table(s, alors l'écran de commande pour eux n'apparaitra pas dans l'application. Par défaut,il peut faire tout ceci. Toutefois, si JMRI n'est pas capable de les contrôler, les régulateurs ne seront pas capables de le faire.

En supplément, vous pouvez limiter quels aiguillages apparaissent. Il y a un élément appelé "Filtre de Contrôles" qui vous permet de dire quels aiguillages afficher sur l'écran ou le cacher.
