L'éditeur Panneau de Commande

Éditeur Panneau de Commande est tout simplement une vue alternative et un contrôleur avec exactement le même contenu de données que dans l'Éditeur Panneau. Vous pouvez modifier l'affichage et les commandes de votre panneau à chaque fois et aussi souvent que vous voulez. Le Stockage et le chargement peut être fait de l'un ou l'autre éditeur. L'éditeur utilisé pour stocker le panneau sera l'éditeur utilisé au prochain chargement.

Note: L'Éditeur de Panneau de Commande a été développé et testé sur Windows uniquement. Le développeur aimerait beaucoup avoir ce travail d'application sur les autres systèmes d'exploitation (Par exemple Mac et Linux), mais est incapable de tester sur ces milieux. Tout commentaire serait apprécié.

Modification de la Vue entre les Éditeurs

Dans l'un ou l'autre des éditeur en mode édition le dernier élément dans le menu Dossier est Modifier vue de "...". ("..." étant l'autre éditeur). Sélectionner cette option de commutation de la vue d'un mode éditeur pour un autre mode éditeur.

Différences dans la Vue entre les Éditeurs

L'Éditeur de panneau a une fenêtre séparée pour ses fonctions d'édition.Pas l'Éditeur Panneau de Commande . A la place trois nouveaux menus sont introduits dans le panneau: Modifier, Options, Zoom, AddItems. Lorsque le mode de fonctionnement (non-edit) est sélectionné ces trois menus disparaissent.

Menu Fichier

Le Fichier menu change ces éléments entre le mode d'édition et le mode de fonctionnement. Dans le mode d'édition les articles sont les mêmes que les éléments dans le menu Fichier de la fenêtre d'édition de l'Éditeur de Panneaux avec quelques articles supplémentaires. Ils sont les suivants:

  • Nouveau Panneau.
  • Panneaux Stockés ....
  • Stockage Index Images.
  • Renommer Panneau.
  • Créer/Modifier Index des images.
  • Changement Vue de l'Éditeur de Panneau.
  • Changer vue du Générateur de Circuit.
  • Supprimer Ce panneau ....
  • Fermer Éditeur.

en mode Éditeur de Panneau de Commande, le menu Fichier n'a qu'un seul élément, Ouvrir Éditeur

Menu Édition

Ce menu soutient Couper, Copier et Coller avec le système presse-papier. Couper et Copier placera une copie d'un article de la sélection ou un groupe dans le presse-papiers. Cette fonctionnalité permet des copies de sélections d'un panneau à un autre panneau.

Il y a également un élément de menu pour sélectionner tous les éléments du panneau.

Options du Menu

Les éléments de ce menu sont similaires aux cases à cocher dans la fenêtre d'édition de l'Éditeur de Panneau avec quelques éléments et caractéristiques supplémentaires.

L'option Ignorer position individuelles et les paramètres de contrôle active ou désactive, les deux prochaines éléments de menu (Tous les éléments du panneau peuvent être repositionnée et Tous les éléments du panneau contrôle le tracé). Lorsque la position dérogation individuelle et de contrôle des paramètres n'est pas cochée, le verrouillage ou déverrouillage de la position des icônes se fait comme dans l'Éditeur de Panneau - comme est activé ou désactivé la commande d'un élément du réseau. Lorsque la position dérogation individuelle et les paramètres de contrôle est cochée, le deux prochains articles de menu fonctionnent comme ils le font dans l'Éditeur de Panneau . Toutefois, le verrouillage individuel et les paramètres de commande pour chaque élément sont mémorisés.

Décocher l'élément position dérogation individuel et les paramètres de contrôle l'élément sera rétablira les paramètres des élément individuel à leurs paramètres précédents. Dans l'Éditeur de Panneau, la vérification Tous les articles du panneau peuvent être repositionnés et Tous les éléments du panneau de commande du réseau fixera définitivement tous les éléments du panneau.

Les définition des préférences de la barre de défilement se trouvent également dans ce menu.

Menu Zoom

L'élément zoom (pourcentage )... va agrandir ou réduire la taille du panneau.

l'élément Zoom pleine Page va agrandir ou réduire l'affichage de tous les éléments du panneau dans le panneau plein écran dans la fenêtre sans l'utilisation de barres de défilement. Cela peut être utile si votre panneau de conception se trouve être légèrement plus grand que ce que vous pouvez voir à l'écran. Plutôt que qu' utiliser les barres de défilement pour voir tous les éléments sur le panneau, "Zoom Plein Écran" permettra de réduire pour adapter l'affichage à la fenêtre du panneau. Inversement, il élargira l'affichage pour s'adapter aux fenêtre du panneau. Si vous redimensionnez la fenêtre du panneau, alors vous devriez répéter "Zoom Plein Écran" pour redimensionner l'écran pour s'adapter aux nouvelles dimensions de la fenêtre du panneau.

Menu Ajt Articles

Le menu Article Palette ouvre une fenêtre à onglets qui vous permet d'ajouter des éléments au panneau. Chaque type d'élément a un onglet dans la fenêtre. L'ajout d'éléments se fait par glisser-déposer.Des éléments iconiques peuvent avoir plusieurs ensembles d'icônes à choisir. Les jeux d'icônes peuvent être personnalisés et ajoutés ou supprimés en fonction de l'utilisateur. Les détails peuvent être trouvés au La Palette article . Notez que l'article de la palette peut également être ouvert avec la touche accélérateur, ctrl P.

La liste des éléments option de menuTableau ouvre la liste des tables appareil. L'élément de menu peut également être appelé avec la touche de raccourci, Ctrl T.

Utilisation de la souris dans l'Éditeur de Panneau de Commande

L'Éditeur Panneau de Commande n'implémente pas la mise en évidence de renversement trouvé dans l'Éditeur de Panneau.

Une autre différence est que la sélection se fait avec un simple clic. Faites un clic droit (méta-cliquez sur sur Mac? ou Linux?) fait apparaître le menu contextuel de l'élément.

Glisser peut être fait soit avec un simple clic ou une méta-cliquez dessus.

Sélection multiple

Comme avec l'Éditeur de Panneau, un groupe de sélection peut être faite en utilisant la «zone de la sélection» - un rectangle en pointillés rouges qui va sélectionner tous les articles complètement au sein de sélection. Si la touche Maj est maintenue enfoncée tout en créant une zone de sélection, alors tous les éléments qui croisent le rectangle en pointillés rouges sont sélectionnés. Cela peut être une fonction très utile lorsque les éléments sont situé à la frontière en haut et à gauche du panneau.

Une autre caractéristique commune est l'ajout ou la suppression d'éléments dans un groupe de sélection par l'appui sur la touche Ctrl quand vous sélectionnez l'article.

Glisser Sélections

Glisser une sélection ou un groupe de sélection peut être faite soit avec bouton de la souris dans l'Éditeur de Panneau de Commande. Une icône de fond

(c'est à dire une icône au niveau 1) peut être déplacée en la faisant glisser avec la touche Ctrl enfoncée.

Duplication Sélections Dans Éditeur Panneau de Commande

La copie d'un élément ou un groupe de sélection est faite par l'utilisation du menu contextuel pour l'un des éléments sélectionnés. Lorsque l'élémentCopier du menu contextuel est cliqué, une copie du groupe de sélection se fait sur le dessus de la sélection originale. Faites glisser cette copie à l'endroit où vous souhaitez placer la copie. La sélection initiale est maintenant désactivée et reste à sa position initiale. La procédure de copie peut être répétée sur la nouvelle position copie pour continuer à faire plus de copies.

Attributs du Texte Edité

Les Attributs du Texte Edité des éléments contextuel ouvre une boîte de dialogue pour définir l'un des attributs de texte de l'élément.

Modification Sélection Multiple

Lorsqu'un élément de menu contextuel est invoqué à partir d'un élément qui est un membre d'un groupe de sélection , toutes les valeurs modifiées qui ont été faites dans le menu contextuel sont appliqués à tous les membres du groupe de la sélection si cette valeur est pertinente pour les membres. Par exemple, si la taille de la police est modifiée sur l'un des éléments du groupe dans la sélection, ensuite la taille de police est modifiée dans tous les éléments du groupe. Si par exemple, une horloge rapide a été incluse dans ce groupe, alors aucune modification ne serait apportée à l'horloge.
