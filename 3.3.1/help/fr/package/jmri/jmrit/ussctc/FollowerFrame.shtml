Suiveurs


Ceci fournit un moyen simple pour définir un aiguillage qui «suit» l'état d'un capteur d'entrée. Vous voudrez peut-être ceci, alors vous pouvez connecter un interrupteur TCO qui bascule à la fois sur un capteurr, et qui pilote un interrupteur de moteur qui est connecté comme un aiguillage JMRI, par exemple,

Les champs et les boutons:

Nom de sortie
Saisissez le nom système ou nom utilisateur de l'aiguillage que le suiveur va contrôler
Nom de la sonde
Entrez le nom système ou nom 'utilisateur du capteur le suiveur se regarder
Inverser
Normalement, l'aiguillage est fixé dévié lorsque le capteur est ACTIF, et ensemble fermé lorsque le capteur est inactif. Vérification que la case "Inverser" entraîne l'aiguillage en position DÉVIÉ lorsque le capteur est INACTIF, et DIRECT lorsque le capteur est ACTIF.
Nom veto
Si un capteur est ajouté ici, le suiveur ne contrôlera pas l'aiguillage lorsque le capteur est ACTIF. Ceci peut être utilisé par exemple pour verrouiller un aiguillage depuis un contrôle local.
Voir
Entrez un nom d'aiguillage de sortie et cliquez sur ce bouton pour voir toute configuration suiveuse existante .
Ajouter/Mettre à jour
Dossier hors champs comme désiré et appuyez sur Ajouter/Mettre à jour pour créer un nouveau suiveur ou en modifier un existant.
