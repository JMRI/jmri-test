JMRI: Fenêtre Principale Window

La fenêtre principale PanelPro vous donne accès à toutes les capacités de JMRI

Aide sur des menus spécifiques:

  • Edit - (Préférences)
  • Outils
  • Liste
  • Panneaux
  • ((Menus spécifiques à votre connexion de mise en page apparaît ici)
  • Debogage
  • Fenêtre - Affiche les fenêtres actuellement disponibles
    • Réduire - réduit la fenêtre actuelle, elle se cache.
    Aide
    • Fenêtre Aide
    • - Ouvre une page d'aide spécifique à la fenêtre courante. Pour la fenêtre principale, c'est cette page.
    • Aide Générale - Ouvre la page d'aide principale JMRI
    • Licence - Affiche la licence JMRI.
    • Emplacements - Montrer où les répertoires et les préférences du programme JMRI sont situés
    • Contexte - Affiche des informations sur votre configuration de JMRI qui pourraient être utiles quand quelqu'un essaie de vous aider à résoudre un problème.
    • Téléchargement Infos de Débogage - Ouvre une fenêtre pour le téléchargement d'info débogage .
