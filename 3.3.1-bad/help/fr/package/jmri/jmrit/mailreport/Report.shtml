JMRI: Téléchargement des Informations de Débogage

L'élément "Transférer les informations de débogage" dans les menus d'aide JMRI vous permet de stocker des informations de débogage, où les développeurs JMRI peuvent facilement les trouver. Les informations contiennent tous les détails spécifiques de votre installation JMRI, y compris le journal de la session la plus récente du programme, mais ne contiennent aucune de vos informations privées. Vous pouvez voir les rapports des problèmes antérieurs dans le jmri-rapports des problèmes au groupe Google .

Pour rendre l'information plus utile, vous devriez télécharger dès que possible après avoir rencontré le problème, de sorte que toute information utile n'ait pas été écrasée.

L'élément «Informations Upload débogage" va aider en ouvrant une nouvelle fenêtre avec trois champs et deux cases à cocher. Entrez votre adresse email dans la ligne du haut. elle ne sera pas rendue visible pour le public, mais nous avins besoin d'une adresse courriel valide pour réduire les spams. Dans le champ suivant, entrez un résumé d'une ligne du problème qui apparaîtra comme le sujet du téléchargement. Enfin, tapez quelques lignes décrivant ce qui s'est mal passé, que vous avez recréer, etc

Normalement, vous laissez les deux cases à cocher afin que le rapport contiendra les informations de contexte (version JMRI, la version Java, types de matériel, etc) et les fichiers journaux les plus récents. Si vous ne voulez pas inclure ceux-ci pour une raison quelconque, il suffit de décocher leur case

Enfin,appuyer sur envoyer. Si tout va bien, la fenêtre va se fermer quelques secondes après que l'information ait été envoyé.
