JMRI: Menu Panneaux

Le menu "Panneaux" JMRi contient les éléments suivants:
  • Nouveau Panneau ...: Sélectionnez cette option pour créer un nouveau panneau dans les deux Éditeur de panneau ou Éditeur de TCO .
  • Chargement de Panneau...: Charge un fichier stockés précédemment avec le choix ci-dessous "Stocker Panneaux ...» . Si le fichier contient Éditeur de Panneaux ou l'Éditeur de TCO, ils seront ajoutés à tout ce qui existe déjà dans le programme.
  • Stocker Panneaux ...: Stocker la configuration actuelle du programme vers un fichier, incluant toutous les Éditeur de Panneau actuel et les Éditeurs de TCO et de Configuration. Utilisez "Montrer Panneau " ci-dessous pour voir la liste complète.
  • Afficher l'historique : Affiche un historique de tous les fichiers chargés dans le programme en cours, et comment ils ont été stockés à partir des exécutions précédentes du programme, avec ce qui a été chargé dans ceux ci, et ainsi de suite. Seuls les dossiers d'information de JMRI 2.9.3 et versions ultérieures.
  • Montrer Panneau : Affiche la liste des TCO et/ou des panneaux des Éditeurs de Panneau actuellement dans le programme, et vous permet d'apporter l'un à l'avant en le sélectionnant.
  • Exécuter le script ...: Sélectionner et exécuter un fichier de script.
  • Moniteur de Lien: Ouvre une fenêtre affichant le lien courant en cours d'exécution , qui comprend l'exécution de scripts long.
  • Sortie de script: Ouvre la fenêtre de sortie du script. Tous les messages à partir de scripts apparaîtront ici.
  • Entrée Script: Ouvre une fenêtre de saisie du script. Vous pouvez taper directement les commandes de script ici.
