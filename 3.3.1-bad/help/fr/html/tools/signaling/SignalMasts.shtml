JMRI: Mâts de Signal

Informations générales sur les mâts de signal JMRI

Les objets JMRI "SignalMast« vous permettent d'afficher des aspects prototypiques sur les signaux. Pour plus d'information, consultez l'Aspect Aspect signalisation .

Les Mâts de signal sont configurés via la table mât de signal Vous pouvez en ajouter de nouveaux avec le bouton "Ajouter ..." sur la table, qui vous emmène à la fenêtre "Ajouter mât de signal" .

De la table, vous pouvez définir des mâts de signaunal individuels avec tous les aspects qu'ils sont capables de d'afficher en cliquant dans la colonne "aspect"
