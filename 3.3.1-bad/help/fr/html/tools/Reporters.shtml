JMRI: Documentation Rapport

Rapport

Un rapport est le moyen JMRI de représenter l'information en retournée par le réseau.

Un rapport peut contenir des informations arbitraires, contrairement à un capteur qui n'est qu'ACTIFf ou INACTIF. Chaque rapport est associé à une source d'informations spécifiques sur le réseau, et fait que les informations périphériques sont disponibles pour le reste de JMRI.

La mise en œuvre initiale est pour le transpondage LocoNet (Digitrax) , où le rapport donne accès à l'information à partir de cartes Digitrax BDL168 avec RX4 href="http://www.digitrax.com/prd_statdec_rx4.php">. Lorsque le BDL168 rapporte qu'un décodeur a été vu dans un canton transpondeur ou n'est pas plus visible dans le canton, le rapport correspondant affichera des messages comme "1234 entrer" et "1234 sortie". Il ya aussi un rapport LocoNet mis en œuvre pour l'affichage des messages de détecteurs Lissy .

Le Tableau Rapport fournit un accès facile au contenu de tous les rapports que JMRI connaît. Il ya aussi un bouton "Ajouter ..." pour créer vos propres Rapports internes; JMRI crée automatiquement Objets Reporter pour les appareils sur le réseau.

Affichage des Valeurs Rapport sur les Panneaux

La valeur d'un rapport peut être affichée sur un panneau via une icône. Pour ce faire:
  • Sur la fenêtre Éditeur Panneau de votre panneau, trouver le "Ajouter Rapport" bouton.
  • Dans la boîte de texte à côté de ce bouton, tapez le nom utilisateur ou système du Rapportr désiré.
  • Cliquez sur le bouton. L'icône Rapport sera placée sur le panneau. Si elle n'est pas visible, ce pourrait être parce que la valeur est actuellement vide; utiliser le tableau Rapport pour changer la valeur à quelque chose qui sera visible.
  • Vous pouvez faire glisser l'icône à l'endroit où vous voulez dans la manière habituelle.
  • Le menu pop-up sur l'icône vous permettra de modifier le formatage.
