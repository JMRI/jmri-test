JMRI: DecoderPro Questions Fréquemment Posées

Cette page est la FAQ de DecoderPro de JMRI. Les éléments obtenus listés ici ont souvent été posés, ils sont également disponibles ailleurs..

Comment puis je obtenire un type de décodeur défini?

Un nouveau décodeur complet nécessite un nouveau ou un fichier de configuration mis à jour. Vous pouvez créer votre propre fichier ou demander sur la mailing list JmriUsers Pour demander à quelqu'un d'en créer un pour vous.

Dans beaucoup de cas, vous pouvez utiliser un décodeur existant qui est "suffisamment proche". Par exemple, un décodeur du même fabricant qui a une taille différente ou Notation actuelle pourrait avoir la même configuration de CVs, ainsi la définition de ce décodeur va très bien fonctionner.

Que dois-je faire avec ce nouveau fichier décodeur?

Pour que DecoderPro utilise la définition d'un type de décodeur dans un nouveau fichier décodeur, le nouveau fichier doit être inclut dans l'index DecoderPro des types de décodeur. Pour faire ceci:

  • Dans votre Répertoire d'installation, vous devez trouver un répertoire nommé "xml". Sous ce répertoire, il doit y en avoir un nommé "decoders". Vous devez mettre ce fichier ici.
  • Maintenant démarrer DecoderPro, et ouvrez le menu "Debogage". Sélectionnez l'option "Reconstruire Index Décodeur" .
  • L'ordinateur va fonctionner une minute ou deux avec la sélection en surbrillance. Aussitôt qu'elle disparait, la nouvelle définition est disponible pour vous.

SVP Soyez sûr que le nouveau fichier de configuration a bien contribué au projet ainsi il sera inclut dans les versions futures et vous n'aurez pas à refaire l'index à chaque fois que vous ferez une mise à jour.

Comment Puis je obtenir les dernières définitions du décodeur?

La façon la plus aisée pour mettre à jour est de télécharger une nouvelle version du programme. Il est peu probable de causer des problèmes et installe automatiquement les dernières versions des contributions à décoderPro.

Les mises à jour se produisent en permanence entre les versions du programme. Comme les fichiers sont des contributions, nous les rendons disponibles sur notre page web Définitions de décodeur collectées Vous pouvez les télécharger depuis là, mais notez que la plupart des navigateurs web ne vous permettrons pas de les voir directement; vous devez les télécharger.

Bien des gens travaillent sur des définitions décodeur, ils mettent souvent leurs définitions dans la zône "Fichiers" du jmriusers Yahoo group. Vous pouvez Les télécharger depuis là. Notez que comme mesure anti-spam vous devrez probablement joindre le groupe pour pouvoir télécharger les fichiers; cela necessite que vous ayez un compte Yahoo (gratuit).

Pourquoi mon LocoBuffer-II ne travaille pas avec Windows XP?

Avec Windows XP, les paramètres de votre port série peuvent ne pas être réglés pour un contrôle matériel du flux.Pour corriger ceci, Clic-droit sur My computer et sélectionnez les Propriétés. Allez à l'onglet Matériel et cliquer sur Device Manager.Ouvrez la section qui dit Ports (COM & LPT). Clic-droit sur le port série attribué à LocoBuffer et sélectionnez Propriétés. Allez à l'onglet Réglage du Port et paramétrer les options à 19200, 8, None,1, Hardware. Cliquer sur OK. Fermer tout. Redémarrer JMRI et vérifiez vos paramètres de connection.
