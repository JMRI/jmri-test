L'utilitaire de configuration JMRI LI101F

L'outil de configuration LI101 est utilisé pour configurer l'LI101F.

En partant du haut:

La première liste déroulante permet de sélectionner l'adresse de l'appareil à utiliser sur le XPressNet. Cette adresse doit être différente de celle de tous les autres périphériques sur votre XpressNet.

La seconde liste déroulante affiche les boites fixant la vitesse du port LI101F. Cette vitesse devrait correspondre à ce que vous avez utilisé pour le "Port Speed" paramètre dans le volet préférences

A partir du côté gauche de l'écran, le bouton "Read from LI101" (Lire depuis LI101), va lire les paramètres actuels de l'LI101F.

Le bouton "Write to LI101"(Écrire dans LI101), écrira les paramètres sur l'écran de l'LI101F. Si vous avez changé le réglage de la vitesse LI101, vous aurez également besoin de changer le réglage de la vitesse dans vos préférences.

Le bouton "Reset to Factory Defaults" (Remettre aux paramètres par défaut) réinitialise l'écran d'usine par défaut. Ceci ne veut pas envoyer une instruction de réinitialisation à l'LI101F à ce moment.

Le dernier bouton, étiqueté "Fermer" va fermer l'écran, sans envoyer aucune information à la LI101F.
