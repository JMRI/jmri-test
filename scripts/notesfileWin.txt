The JMRI 2.7.2 test version is now available for download. 

This is the specific version for Windows users. 

It contains many new features, including:
* fix for the problem JMRI 2.7.1 to occasionally hang
* improvements to Logix and Panel Editor
along with other changes.

For more information, please see the release note at 
http://jmri.sf.net/releasenotes/jmri2.7.1.shtml
